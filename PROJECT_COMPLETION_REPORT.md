## 🚀 使用指南

### 启动应用
```bash
cd /Users/huzhiheng/code/retas-ageless
cargo run
```

### 基础绘制
1. 选择 Brush 工具 (工具栏)
2. 在画布上绘制
3. 调节笔刷大小 (右侧面板滑块)
4. 选择颜色 (颜色选择器)

### 矢量绘图
1. 选择 Pen 工具
2. 在 Vector Layers 面板确认模式 (Create/Edit)
3. Create 模式: 点击创建路径点
4. Edit 模式: 拖拽控制点编辑

### 上色
1. 选择 Fill 工具
2. 选择填充模式 (Normal/Smart/Gap Closing)
3. 点击闭合区域填充
4. 调节容差滑块控制精度

### 应用效果
1. 打开 Effects 面板
2. 点击 Blur/Glow/Brightness 添加效果
3. 点击效果开关启用/禁用
4. 调节参数滑块

### 批量导出
1. 打开 Export 面板
2. 选择格式 (PNG/JPEG/GIF/MP4)
3. 选择质量 (Draft/Standard/High)
4. 选择帧范围 (All/Current)
5. 点击 Export Current Project
