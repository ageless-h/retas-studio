### 刪除描繪內容的工具

刪除描繪內容的工具刪除描繪內容的工具
介紹如何使用[橡皮擦]工具或透明色，在CLIP STUDIO PAINT中擦除畫布上的繪畫。
橡皮擦工具
[工具]面板上的[橡皮擦]工具可以刪除已繪製的內容。
除了可以刪除所選圖層之外，還能使用[穿透圖層]輔助工具來刪除所有圖層，或利用[向量線用]輔助工具來刪除向量圖層的整條線條等，根據所選的輔助工具和圖層搭配組合，能運用各種效果。

此外，使用[工具屬性]面板和[輔助工具詳細]面板，可以進行更詳細的設定。
關於可以在[工具屬性]面板和[輔助工具詳細]面板中進行的設定，請參照『筆刷的設定項目』。
向量圖層【PRO／EX】
使用[橡皮擦]工具刪除向量圖層的描繪，看起來描繪像是被刪除了，但其實是在使用[橡皮擦]工具塗抹的地方產生透明的線條。
使用[向量線用]輔助工具，只需塗抹線的一部分，即可刪除塗抹部分到繪製在向量圖層上其他線的交點之間的線條。詳情請參照『刪除向量線』。

以透明色刪除
使用[鉛筆]工具或[毛筆]工具等筆刷，以透明色進行描繪時，可以作為橡皮擦使用並完整保留筆刷觸感。
透明色可以透過點擊[工具]面板或彩色面板上的格紋區塊來選擇。

### 筆刷尺寸面板

筆刷尺寸面板筆刷尺寸面板
CLIP STUDIO PAINT的[筆刷尺寸]面板可以從預設的數值中選擇描繪系的工具，如：[沾水筆]工具或[毛筆]工具，以及[圖形]工具等的筆刷尺寸。另外，也可以登記常用的筆刷尺寸。

筆刷尺寸面板的自訂設定
預設的追加與刪除
[筆刷尺寸]面板中登記的筆刷尺寸被稱為「預設」。
欲將目前使用的筆刷尺寸登記為[筆刷尺寸]面板的預設筆刷尺寸時，可以在[筆刷尺寸]面板的選單中選擇[將當前尺寸追加到預設中]。
欲刪除[筆刷尺寸]面板中的預設時，請從[筆刷尺寸]面板的選單中選擇[刪除當前尺寸的預設]。
顯示滑桿
可以在[筆刷尺寸]面板上顯示調整筆刷尺寸的滑桿。從[筆刷尺寸]面板的選單中選擇[顯示滑桿]，可以切換滑桿的顯示或隱藏。

### 調節筆壓檢測等級

調節筆壓檢測等級調節筆壓檢測等級
執行CLIP STUDIO PAINT的[調節筆壓檢測等級]，可以連接繪圖板或附帶的感應筆，來調整筆壓以符合使用者期許的手感。[調節筆壓檢測等級]可以透過以下操作來執行。
·Windows：選擇[檔案]選單→[調節筆壓檢測等級]。
·macOS／平板電腦版：請選擇[CLIP STUDIO PAINT]選單→[筆壓檢測等級]。
[調整筆壓]對話方塊顯示後，請使用平時的筆壓，並刻意加上強弱變化，在畫布上畫一次線。
若對所畫的線條的強弱變化沒有到滿意，可以再次嘗試繪畫線條。描繪完成後，點擊[確認調整結果]。

[確認調整]對話方塊顯示後，請使用平時的筆壓，在畫布上進行試畫。
若想要調整繪畫的筆觸，請點選[更堅硬]或[更柔軟]，並再次於畫布上試畫，以調整感覺。

當筆觸符合理想狀態時，請點擊[確認調整]對話方塊的[結束]。
使用圖表進行設定
將[調整筆壓]對話方塊和[確認調整]對話方塊上的[顯示筆壓曲線圖]設定為開啟後，可以顯示筆壓曲線圖。使用筆壓圖表，可以詳細設定繪畫時的筆觸。

可以使用圖表調整筆壓檢測等級。拖拽圖表的曲線，可以更詳細地調整筆的硬度。

向量圖層【PRO／EX】
使用[橡皮擦]工具刪除向量圖層的描繪，看起來描繪像是被刪除了，但其實是在使用[橡皮擦]工具塗抹的地方產生透明的線條。
使用[向量線用]輔助工具，只需塗抹線的一部分，即可刪除塗抹部分到繪製在向量圖層上其他線的交點之間的線條。詳情請參照『刪除向量線』。
