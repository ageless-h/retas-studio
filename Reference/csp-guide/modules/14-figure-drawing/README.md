# 使用圖形進行描繪
介紹如何以CLIP STUDIO PAINT的[圖形]工具繪製圖形。可繪製[直線]、[曲線]、[折線]、[貝茲曲線]、[多邊形]、[橢圓]等圖形。
·​直線輔助工具
·​曲線輔助工具
·​折線輔助工具
·​連續曲線輔助工具・貝茲曲線輔助工具
·​長方形輔助工具・橢圓輔助工具
·​多邊形輔助工具
- 使用圖形進行描繪

## 详细内容

### 直線輔助工具

直線輔助工具直線輔助工具
使用[圖形]工具→[直接描繪]的[直線]輔助工具，可以在畫布上進行拖拽來繪製直線。按住[Shift]鍵拖拽，在初始設定下，能以45°的間隔描繪直線。

[直線]輔助工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定，可以變更圖形的消除鋸齒、筆刷前端形狀、角度刻度等。
在[輔助工具詳細]面板中，可以透過『單位曲線』的類別設定曲線的描繪方法。除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。

### 曲線輔助工具

曲線輔助工具曲線輔助工具
使用[圖形]工具→[直接描繪]的[曲線]輔助工具，可以在畫布上進行拖拽，繪製出單一的曲線。
從曲線的起始位置開始拖拽到結束位置，並移動滑鼠以調整曲線的彎曲程度，點擊後即可確定曲線。
若使用感應筆或手指進行繪畫，在拖拽至曲線的結束位置後，可於畫布上的另一個位置進行拖拽來調整曲線的彎曲程度。

[曲線]輔助工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定，可以變更圖形的消除鋸齒和筆刷前端形狀等。
在[輔助工具詳細]面板中，可以透過『單位曲線』的類別設定曲線的描繪方法。除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。
3次貝茲
若選擇[輔助工具詳細]面板，[單位曲線]→[曲線]中的[3次貝茲]，可以使用不同於初始設定的方式來繪製曲線。
從曲線的起始位置開始拖拽到結束位置後，可以使用2個控制點來調整曲線的彎曲程度。
移動滑鼠游標以調整曲線的彎曲程度，並進行點擊。進一步移動滑鼠游標，調整曲線的彎曲程度。
若使用感應筆或手指進行繪畫，可以在從曲線的起始位置開始拖拽到結束位置後，於畫布上拖動2個控制點，調整曲線的彎曲程度。

### 折線輔助工具

折線輔助工具折線輔助工具
使用[圖形]工具→[直接描繪]的[折線]輔助工具，可以繪製通過多個點的折線。
點擊畫布，指定折線的通過點。雙擊後，將確定描繪內容。點擊畫布視窗以外的畫面，也能確定描繪。
若在最後點擊最初點擊的點，將確定描繪，並建立1個閉合的圖形。

備忘

在描繪折線的途中，若想要編輯線條或點，請參考『折線和曲線的編輯方法』。

[折線]輔助工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定，可以變更圖形的消除鋸齒和筆刷前端形狀等。
在[輔助工具詳細]面板中，可以透過『連續曲線』的類別設定曲線的描繪方法。除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。

### 長方形輔助工具・橢圓輔助工具

長方形輔助工具・橢圓輔助工具長方形輔助工具・橢圓輔助工具
[圖形]工具→[直接描繪]的[長方形]輔助工具和[橢圓]輔助工具，兩者可以繪製的圖形不同，但基本操作方法相同。
使用[長方形]輔助工具，在畫布上往斜角拖拽，可以描繪長方形。按住[Shift]鍵並進行拖曳，可以繪製出正方形。

使用[橢圓]輔助工具，在畫布上往斜角拖拽，可以描繪橢圓形。描繪橢圓形後移動滑鼠，可以調整橢圓的角度。點擊後，會確定橢圓的描繪。按住[Shift]鍵並進行拖曳，可以繪製出正圓。

[長方形]輔助工具和[橢圓]輔助工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定後，可以變更填充方式和長寬比等。
在[輔助工具詳細]面板中，可以透過『圖形』和『圖形操作』的類別設定圖形的描繪方法。除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。

### 多邊形輔助工具

多邊形輔助工具多邊形輔助工具
使用[圖形]工具→[直接描繪]的[多邊形]輔助工具，可以在畫布上拖拽來描繪多邊形。描繪多邊形後移動滑鼠，可以調整多邊形的角度。點擊後，會確定圖形的描繪。
初始設定為六角形，但只要點擊[工具屬性]面板中[圖形]的[+]，並指定[多邊形的頂點數]，即可更改多邊形的頂點數量。

[多邊形]輔助工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定後，可以變更填充方式和角的弧度等。
在[輔助工具詳細]面板中，可以透過『圖形』和『圖形操作』的類別設定圖形的描繪方法。除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。
