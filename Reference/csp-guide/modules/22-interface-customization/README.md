### 工具配置的自訂設定

工具配置的自訂設定工具配置的自訂設定
使用拖拽操作來移動CLIP STUDIO PAINT的工具和輔助工具。首次進行操作時，請從[工具]面板或[輔助工具]面板的選單中，確認[排序方法]的設定。
·若設定為[拖拽]，只需拖拽即可移動工具或輔助工具。此為初始設定。
·若設定為[Ctrl+拖拽]，按住[Ctrl]鍵並拖拽，即可移動工具或輔助工具。能防止意外移動工具或輔助工具。

工具面板的自訂設定
變更工具名稱
可以使用以下任一方法，更改工具名稱。
·從想要更改名稱的工具上顯示關聯選單，選擇[設定工具]。
·選擇要更改名稱的工具後，從[工具]面板的選單中選擇[設定工具]。
顯示[設定工具]對話方塊後，在[名稱]中輸入工具名稱，然後點擊[OK]。[工具]面板上會套用所輸入的工具名稱。
變更工具的配置
從[工具]面板中點擊想要變更配置的工具，將其拖拽至移動目標上方。

### 隱藏標題列和選單列

隱藏標題列和選單列隱藏標題列和選單列
可以透過隱藏CLIP STUDIO PAINT的標題列和選單列來擴大畫布的繪畫空間。
隱藏標題列【Windows／macOS】
在顯示標題列和選單列的狀態下，選擇[視窗]選單→[隱藏標題列]。將隱藏標題列，並將視窗的顯示最大化。
按下[Shift]+[Tab]鍵，也可以進行相同的操作。

隱藏選單列【Windows／平板電腦】
在顯示選單列的狀態下，選擇[視窗]選單→[隱藏選單列]。將隱藏選單列。Windows的情況下，請先隱藏標題列，再隱藏選單列。
按下[Shift]+[Tab]鍵，也可以進行相同的操作。

將標題列和選單列返回原本的顯示
隱藏標題列和選單列後，若要返回原本的顯示，請執行以下操作。

·若是平板電腦時，請點選命令列的[顯示應用程式選單]，從顯示的選單中選擇[顯示選單列]。若為支援觸控手勢的Windows平板電腦，也可進行相同操作將畫面返回原本的顯示。
