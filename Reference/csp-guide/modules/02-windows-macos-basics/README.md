# Windows／macOS的基本操作
介紹使用Windows和macOS的CLIP STUDIO PAINT繪畫時，從建立畫布、繪製線條，到保存作品的簡單功能。

### Windows／macOS的介面

Windows／macOS的介面Windows／macOS的介面
在Windows／macOS系統中首次啟動CLIP STUDIO PAINT時，會顯示以下畫面。

①標題列顯示正在編輯的畫布檔案名稱、尺寸、解析度、畫布顯示倍率、應用程式名稱等資訊。
②命令列點擊圖示，即可使用各種功能。可以自訂設定，例如顯示常用功能的圖示等。詳細的操作方法請參照『設定命令列【PRO／EX】』 。
③面板Dock儲存多個面板的區域。可以建立在視窗的左右和下方。
詳細操作方法請參照『面板』 。
④面板進行各種設定的畫面，有各種不同種類的面板。畫面中的面板雖然收納在面板Dock內，但也可以像視窗和對話方塊一樣單獨顯示。面板的操作請參照『面板』。
⑤畫布視窗圖像編輯區域。在這裡能進行描繪。

備忘

當使用具有觸控手勢功能的Windows平板電腦時，顯示的介面會與iPad及Android平板電腦的工作室模式相同。詳情請參照『工作室模式的介面』。
若想顯示為和一般Windows相同的介面，可在[環境設定]對話方塊中的『介面』進行設定。

### Windows／macOS的基本操作

Windows／macOS的基本操作Windows／macOS的基本操作
介紹CLIP STUDIO PAINT的操作方式，包括：選擇繪畫工具的方法，以及保存創作作品的方法等。
選擇描繪的工具 
從[工具]面板中選擇要使用的繪圖工具。這裡選擇[沾水筆]工具。

從[輔助工具]面板中選擇沾水筆的種類。

在畫布上執行拖拽，即可用[沾水筆]工具於畫布進行描繪。

描繪所需的工具
CLIP STUDIO PAINT的[工具]面板中，除了[沾水筆]工具外，還有各種用於繪畫的工具。選取工具後，可在[輔助工具]面板中選擇各種筆刷。
關於各工具的特徵，請參照『線條或填充用工具』。

選擇顏色
選擇用筆刷繪製的顏色時，請點擊[工具]面板下方的彩色圖示。能夠從主顏色、輔助顏色和透明色中選擇要繪製或設定的顏色。
關於描繪色的相關內容，請參照『工具面板』。

變更顏色時，要使用[色環]面板。拖拽外圈的色相環選擇大致的顏色。拖拽中央的色彩區域的圓，即可選擇描繪色。選擇的顏色會顯示在色環左下方的色彩圖示上。

 點擊面板上方的索引標籤，可以切換為『色板面板』・『顏色滑桿面板』・『色彩混合面板』・『顏色歷史記錄面
