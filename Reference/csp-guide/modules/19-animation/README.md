### 透光桌圖層的操作

透光桌圖層的操作透光桌圖層的操作
介紹CLIP STUDIO PAINT的透光桌圖層的操作方法。
將膠片登記至透光桌
將想要作為繪圖參考的膠片或圖層，登記在[動畫膠片]面板上，可以將其作為透光桌圖層使用。
將想要在[圖層]面板中作為繪圖參考的膠片或圖層，拖放至[動畫膠片]面板的[膠片固有透光桌]或[畫布共通透光桌]上。
圖層可以拖放到[膠片固有透光桌]或[畫布共通透光桌]兩者中。但如果沒有設定[編輯對象膠片]，就無法在[膠片固有透光桌]上進行拖放。

在[動畫膠片]面板中，會將其登記為透光桌圖層。
如果登記的圖層包含圖層資料夾或圖層蒙版，在[動畫膠片]面板中，圖層資料夾和圖層蒙版將以合併的狀態進行登記。

備忘

·透光桌圖層登記後的顯示顏色和不透明度，可以在「環境設定」對話方塊中設定。詳細請參照『動畫』。
·當拖放圖層時，如果會不小心切換[編輯對象膠片]，可以先設定『將當前的動畫膠片固定為編輯對象』，再登記透光桌圖層，這樣[編輯對象膠片]就不會發生切換。
·登記為透光桌圖層的圖層，無法從[圖層]面板中刪除。請先解除透光桌圖層後，再刪除。關於解除方法，請參照『解除特定的透光桌圖層』。


### 軌道的操作

軌道的操作軌道的操作
介紹CLIP STUDIO PAINT的軌道功能。可以交換軌道的順序，改變膠片的重疊順序，並且讀取音頻檔案和影片檔案。
變更軌道的順序
在[時間軸]面板上，可以變更軌道的排列順序。
選擇要移動的軌道，將其拖放至想要移動的位置。若為平板電腦，請將手指或感應筆放在軌道名稱右側的握把上，並進行拖放。

當軌道的順序改變時，[圖層]面板中的圖層重疊順序也會改變。
如同[圖層]面板一樣，將軌道拖放至圖層資料夾中，即可將軌道收納至圖層資料夾內。但無法將其他軌道收納至動畫資料夾中。

### 動畫資料夾和膠片

動畫資料夾和膠片動畫資料夾和膠片
CLIP STUDIO PAINT的動畫資料夾，是用來管理動畫膠片的資料夾。收納在動畫資料夾中的圖層和圖層資料夾，會被視為替動畫追加動態效果的「膠片」。

在[時間軸]面板上指定膠片，即可建立動畫。

備忘

在時間軸無效的情況下，動畫資料夾會被視為普通的圖層資料夾。膠片也會被視為圖層資料夾內的圖層。

建立動畫資料夾
若想建立動畫資料夾，請執行以下任一操作。
·點擊[時間軸]面板的[新建動畫資料夾]。
·選擇[動畫]選單→[動畫用新圖層]→[動畫資料夾]。
關於動畫資料夾，在[圖層]面板中會追加為動畫資料夾，在[時間軸]面板中則會追加為軌道。

備忘

動畫資料夾可以在圖層資料夾內建立，但無法在動畫資料夾內建立。

動畫資料夾的命名規則
將動畫資料夾的名稱設定為一個字母，在相同畫布中新建動畫資料夾時，會自動將名稱設定為連續的字母。
例如，在建立了動畫資料夾「A」之後，新追加的動畫資料夾的名稱將為「B」。在動畫資料夾「B」和「C」之間新建動畫資料夾時，會在前一個字母後面追加小寫字母，例如「Ba」。
關於更改動畫資料夾名稱的方法，請參照『變更圖層名稱』。

### 片段的操作

片段的操作片段的操作
片段是動畫的編輯單位。在CLIP STUDIO PAINT中，只有在配置了片段的部分，才會顯示膠片和圖層。
選擇片段
請在[時間軸]面板上點擊想選取的片段的上方。將滑鼠游標移動到可以選擇片段的範圍時，滑鼠游標的形狀會變成手掌。
邊按住[Ctrl]鍵邊點擊，可以選擇多個片段。

變更片段的位置
選擇想要移動的片段後，將滑鼠游標移動到片段的上方，再拖放至想要移動的位置。片段的位置將會變更為拖拽後的位置。

選擇多個片段，可以一次移動多個片段。即使選擇了不同軌道的片段，也可以同時移動。但無法將片段移動到其他軌道上。
裁切片段
將滑鼠游標移動到片段的上方並左右拖拽，可以調節片段的長度。將滑鼠游標移動到可以更改長度的位置，滑鼠游標的形狀會變成左右箭頭。

片段內的指定膠片位置不會改變，但如果將片段的長度縮短，超出片段範圍的指定膠片將被刪除。若為音頻軌道和影片軌道的片段，音頻和影片可能會從途中開始或發生中斷。

### 主畫格的操作

主畫格的操作主畫格的操作
使用主畫格功能，除了膠片或圖層的顯示資訊，還能將攝影機角度的位置和音頻檔案的音量等設定資訊記錄至時間軸。
如果在兩個位置上設定了主畫格，會從登記在第一個主畫格中的資訊，變更為下一個主畫格中設定的資訊。例如，如果使用2D攝影機資料夾軌道，能以主畫格進行設定，從左到右移動攝影機製作運鏡。
將主畫格的編輯設為有效
針對選擇的軌道，可以設定主畫格。
從[時間軸]面板中，選擇要編輯主畫格的軌道，並執行以下任一操作。
·在[時間軸]面板上點選[將圖層的主畫格有效化]。
·選擇[動畫]選單→[軌道編輯]→[將圖層的主畫格有效化]。

針對選擇中的圖層或軌道，可以使用主畫格進行編輯。
使用[物件]輔助工具選擇圖層或軌道後，可以進行使用主畫格的編輯，[時間軸]面板和[圖層]面板的顯示也會改變。
·在[時間軸]面板的軌道上，會追加[變形]和[不透明度]等主畫格專用的設定項目。有設定蒙版的軌道，還會追加[蒙版]。點擊軌道名稱左側的[詳細]（+），可以顯示更詳細的項目。
·在[圖層]面板上會顯示無法編輯圖像的鎖定圖示。動畫資料夾內的膠片會顯示半透明的鎖定圖示。
