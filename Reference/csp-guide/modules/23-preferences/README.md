·若為macOS／平板電腦版，請選擇[CLIP STUDIO PAINT]選單→[環境設定]。
選擇[環境設定]對話方塊左側的類別，即可切換設定項目。

### 隱私權設定

隱私權設定隱私權設定
[隱私權設定]可以進行有關CLIP STUDIO PAINT品質改善的資訊傳送設定。[隱私權設定]的畫面會在初次啟動時顯示，但也可以透過以下操作再次顯示。
·若為Windows，請選擇[檔案]選單→[隱私權設定]。
·若為macOS／平板電腦版，請選擇[CLIP STUDIO PAINT]選單→[隱私權設定]。
最初顯示的畫面上，點擊[下一步]後，將顯示[隱私權設定]對話方塊的設定畫面。可以設定[支援程式錯誤的改善]和[參與提升產品品質]。詳情請參照[隱私權設定]對話方塊的說明。點擊[OK]，可以確定隱私權設定。

### 捷徑鍵設定

捷徑鍵設定捷徑鍵設定
[捷徑鍵設定]對話方塊中，可以為主選單・工具屬性・筆刷尺寸等部分項目以及工具・輔助工具設定捷徑鍵。
[捷徑鍵設定]對話方塊可以透過以下操作進行顯示。
·若為Windows中，請選擇[檔案]選單→[捷徑鍵設定]。
·若為macOS／平板電腦，請選擇[CLIP STUDIO PAINT]選單→[捷徑鍵設定]。
捷徑鍵的設定方法
1從[設定區域]中選擇要設定的項目。
根據選擇項目，將切換捷徑鍵一覽的顯示。

2從捷徑鍵一覽中選擇要設定捷徑鍵的項目後，點擊[編輯捷徑]。

3按下想要設定為捷徑鍵的按鍵。
·若為平板電腦，也可以設定為側邊鍵盤的按鈕。在進行設定之前，請顯示側邊鍵盤並點擊想要分配的按鍵。
·若使用的是Wacom製的觸控筆，也可以設定觸控筆的側邊按鈕。
·若為macOS，對於已於[系統偏好設定]中設定的鍵盤捷徑鍵，無法設定重複的捷徑鍵。

4點擊[捷徑鍵設定]對話方塊中按鈕以外的地方，可以確定輸入的捷徑鍵。按下[Enter]鍵也能進行確定。
若想要取消輸入的內容，請在確定捷徑鍵之前按下[Esc]鍵。
5點選[捷徑鍵設定]對話方塊的[OK]後，可以使用設定的捷徑鍵

### 設定修飾鍵【PRO／EX】

設定修飾鍵【PRO／EX】設定修飾鍵【PRO／EX】
在[設定修飾鍵]對話方塊中，可以將[Ctrl]・[Alt]・[Shift]・[Space]各個按鍵，及作為組合鍵的滑鼠滾輪・點選右鍵・尾部開關・觸控手勢等，分配給工具的暫時變更和筆刷尺寸變更等操作。
[設定修飾鍵]對話方塊可以透過以下操作進行顯示。
·若為Windows，請選擇[檔案]選單→[設定修飾鍵]。
·若為macOS／平板電腦，請選擇[CLIP STUDIO PAINT]選單→[設定修飾鍵]。
設定修飾鍵
1從[共通設定]和[按照工具的處理進行設定]中，進行選擇。此處選擇[共通設定]。

2從[限定範圍]中選擇要分配設定的按鍵。若選擇[Ctrl]，只會顯示使用[Ctrl]鍵的操作。

3從下拉選單中選擇要設定的修飾鍵操作。這裡選擇[臨時變更工具]。

4當顯示[設定臨時變更工具]對話方塊時，請選擇要切換的工具後，點擊[OK]。

5選擇的工具將顯示在[設定修飾鍵]對話方塊中。

6點擊[設定修飾鍵]對話方塊中的[OK]，可以啟用修飾鍵的設定。
設定修飾鍵對話方塊

①共通設定若要設定可以在所有工具中使用的修飾鍵操作，請選擇此

### 自動動作面板【PRO／EX】

自動動作面板【PRO／EX】自動動作面板【PRO／EX】
CLIP STUDIO PAINT的[自動動作]面板，可以管理記錄有多個操作的自動動作。使用自動動作，可以透過一次點擊執行多個操作。

①切換組請參照『切換自動動作組』。
②新建組請參照『建立自動動作組』。
③追加自動動作組可以讀取從CLIP STUDIO ASSETS下載的自動動作組素材。詳情請參照『讀取素材的方法』。
④自動動作一覽在自動動作組中登記的自動動作一覽。

備忘

長按或右鍵點擊自動動作一覽，可以顯示關聯選單，從選單中可以進行自動動作和命令的編輯。

⑤自動動作用於記錄和再現複數操作的設定。可以再現每個自動動作的操作。點選自動動作名稱左側的圖示，可以切換顯示或隱藏自動動作所記錄的操作。
⑥命令記錄在自動動作中的操作。點擊左側的圖示，將顯示在自動動作中記錄操作時以對話方塊設定的內容。
⑦執行切換方格點擊來關閉設定，可以使自動動作中的特定命令不再再現。
⑧設定切換方格點擊來開啟設定，當再現命令時，將顯示對話方塊以進行設定。設定對話方塊後，將繼續再現自動動作。設為開啟時，會執行紀錄至自動動作時設定的內容。

### 關於CLIP STUDIO PAINT的啟動初始化

關於CLIP STUDIO PAINT的啟動初始化關於CLIP STUDIO PAINT的啟動初始化
若在使用CLIP STUDIO PAINT時遇到故障或其他問題，可以將設定初始化來啟動程式。顯示[啟動初始化]對話方塊，選擇要初始化的項目後，點選[OK]，會刪除選擇的項目設定，恢復為初始設定。

重要

當執行設定初始化時，至今的設定將無法使用。

在進行初始化啟動之前，請先結束CLIP STUDIO PAINT。若為iPad／Android，也請結束背景執行。
[啟動初始化]對話方塊可以透過以下方式進行顯示。
Windows／macOS
啟動CLIP STUDIO，按住[Shift]鍵並點擊[PAINT]，會顯示[啟動初始化]對話方塊。
iPad
依序點擊OS的[設定]→[Clip Studio]，將[啟動初始化]設為開啟。下次啟動CLIP STUDIO PAINT時，會顯示[啟動初始化]對話方塊。
Android
從主畫面上長按Clip Studio的圖示。當顯示選單時，請點擊[啟動初始化]。下次啟動CLIP STUDIO PAINT時，會顯示[啟動初始化]對話方塊。

捷徑鍵設定捷徑鍵設定
[捷徑鍵設定]對話方塊中，可以為主選單・工具屬性・筆刷尺寸等部分項目以及工具・輔助工具設定捷徑鍵。
[捷徑鍵設定]對話方塊可以透過以下操作進行顯示。
·若為Windows中，請選擇[檔案]選單→[捷徑鍵設定]。
·若為macOS／平板電腦，請選擇[CLIP STUDIO PAINT]選單→[捷徑鍵設定]。
捷徑鍵的設定方法
1從[設定區域]中選擇要設定的項目。
根據選擇項目，將切換捷徑鍵一覽的顯示。

2從捷徑鍵一覽中選擇要設定捷徑鍵的項目後，點擊[編輯捷徑]。

3按下想要設定為捷徑鍵的按鍵。
·若為平板電腦，也可以設定為側邊鍵盤的按鈕。在進行設定之前，請顯示側邊鍵盤並點擊想要分配的按鍵。
·若使用的是Wacom製的觸控筆，也可以設定觸控筆的側邊按鈕。
·若為macOS，對於已於[系統偏好設定]中設定的鍵盤捷徑鍵，無法設定重複的捷徑鍵。

4點擊[捷徑鍵設定]對話方塊中按鈕以外的地方，可以確定輸入的捷徑鍵。按下[Enter]鍵也能進行確定。
若想要取消輸入的內容，請在確定捷徑鍵之前按下[Esc]鍵。
5點選[捷徑鍵設定]對話方塊的[OK]後，可以使用設定的捷徑鍵
