### 將姿勢添加到3D素描人偶或3D角色素材中

CLIP STUDIO PAINT的3D素描人偶和3D人物素材可以添加姿勢。

讀取姿勢素材
[素材]面板中的[3D]→[姿勢]中，收錄了可用於3D素描人偶和3D人物素材的姿勢素材。姿勢素材有三種類，分別為全身的姿勢、手部的姿勢，以及連號姿勢。

備忘

選擇3D素描人偶和姿勢素材的同時，會新建3D圖層。詳情請參照『選擇3D素描人偶和姿勢素材進行創作』。

用[物件]輔助工具選擇在畫布上配置的3D素描人偶或3D人物素材。

可以使用以下任一方法來讀取姿勢素材。
由物件啟動器進行讀取
在所選的3D素材上，點擊物件啟動器中的[將姿勢素材套用至模組]，會顯示『追加素材對話方塊』。選擇要使用的姿勢素材後，3D素材上會套用姿勢。

由工具屬性面板進行讀取
從[工具屬性]面板的[姿勢]中點擊[將姿勢素材套用至模組]。會顯示『追加素材對話方塊』。選擇要使用的姿勢素材後，3D素材上會套用姿勢。

從素材面板讀取
從[素材]面板中選擇姿勢素材，點擊[貼上素材]。

### 讀取3D檔案

讀取3D檔案讀取3D檔案
介紹在CLIP STUDIO PAINT的畫布上讀取FBX或OBJ等通用的3D檔案的方法，以及讀取[素材]面板中的3D素材的方法。
讀取3D素材   
從[素材]面板中讀取3D素材到畫布時，請從[素材]面板中選擇要讀取的3D素材，並執行以下任一操作。
·將3D素材拖拽到畫布上。
·點選[素材]面板的[貼上素材]按鈕。詳情請參照『讀取素材的方法』。
·將3D素材拖拽到[圖層]面板上。
·將3D素材拖拽至『四面圖面板【EX】』上。

當3D素材被讀取到畫布上時，同時會新建一個3D圖層。
建立3D圖層的同時，也會建立符合3D空間的3點透視『透視尺規』。建立3D圖層後，透視尺規會先被設定為隱藏。

保持選擇新建的3D圖層的狀態並讀取另一個3D素材，即可在同一個3D圖層上讀取多個3D素材。
第二個以後的3D素材，可從[素材]面板中拖拽並放置在指定位置。

全景素材無法進行多個配置。將全景素材讀取到已配置全景素材的3D圖層中，即可替換原先的全景素材。

### 設定光源和陰影

設定光源和陰影設定光源和陰影
可以在CLIP STUDIO PAINT的3D圖層上設定整體的光源和陰影顯示方式。
將光源套用在3D素材上
以[物件]輔助工具選取3D圖層後，進行以下操作，即可將光源套用到3D圖層中。
如欲顯示整個3D圖層的陰影設定，請從[輔助工具詳細]面板的『環境』類別中，顯示[渲染設定]對話方塊。
從[渲染設定]對話方塊的[3D圖層設定]中的[照明]，將[受光源影響]設為開啟。

在3D圖層中，可以設定以下的光線。
平行光1・平行光2
平行光是從特定方向照射3D素材的光線，可以調整光源的方向、光的顏色和強度。
平行光有兩種類型，分別是[平行光1]和[平行光2]，可以從多個方向照亮3D素材。通常情況下，[平行光2] 被設定為隱藏。
環境光
環境光是從所有方向均勻照射在3D素材上的光線。可以調整光線的顏色和強度。環境光能以其他的圖像或全景素材為基準進行設定。詳情請參照『設定環境光』。
將光源套用在每個3D素材上
以[物件]輔助工具選取3D素材後，進行以下任一操作，即可將光源套用到所選擇的3D圖層中。

### 在3D素材描繪

在3D素材描繪在3D素材描繪
可在3D素描人偶與Ver. 2以上版本3D基礎圖形上，以[沾水筆]工具等筆刷進行描繪。

備忘

·[毛筆]工具的[點畫]輔助工具與[裝飾]工具的[亮片]筆刷等，設定有[顏色變化]和[底子混色]的輔助工具，描繪時將無法套用此設定。
·無法使用[沾水筆]工具的[點筆]輔助工具。

以[物件]輔助工具選擇想描繪的3D素描人偶或3D基礎圖形。切換為[毛筆]工具等，在所選擇的3D素描人偶或3D基礎圖形上描繪。描繪的內容會套用至畫布上顯示的部分。

若想在未顯示的地方描畫時，切換為[物件]輔助工具後，執行3D素材的旋轉或3D圖層整體的攝影機角度變更，即可顯示想描畫的部分。若為3D素描人偶，還可以變更姿勢等。

也可描畫在已讀取材質的3D基礎圖形上。[混合模式]等重疊塗抹時的設定不會套用到材質上。
以筆刷描繪時的材質，請在『材質詳細資訊』類別將[放大率]設為「100.0」，[位置]設為「0.0」。設定為上述數值以外時，會無法進行描繪。

### 編輯3D人物

編輯3D人物編輯3D人物
可以使用預設或造型鍵變更表情，將3D人物登記為素材。
變更表情
以[物件]輔助工具選擇3D人物素材時，可以變更人物的表情。設定方法會因3D人物素材而異。

備忘

使用表情設定

### 執行編輯全景素材

執行編輯全景素材執行編輯全景素材
CLIP STUDIO PAINT的全景素材可以使用[物件]輔助工具[輔助工具詳細]面板進行顯示調整，並且可以進行圖像的讀取和寫出等操作。

調整全景素材的顯示
全景素材的顯示範圍可以在[工具屬性]面板和[輔助工具詳細]面板的以下項目中進行調整。
魚眼透視

在[工具屬性]面板和[輔助工具詳細]面板的『攝影機』類別設定[透視]，可以調整整個3D圖層的透視，並改變顯示範圍。
同一個3D圖層中的所有物件都會受到影響，若數值設定得太大，可能會導致其他3D素材的顯示變形。

### 3D數據的種類

3D數據的種類3D數據的種類
介紹關於CLIP STUDIO PAINT可使用的3D素材。其它關於能夠讀取到3D圖層的3D檔案，請參照 『讀取3D檔案』。
3D素描人偶
3D素描人偶可以調整體型和姿勢。3D素描人偶中設定的體型，可以保存為素材。收錄在[素材]面板的[體型]中。

3D頭部模組
3D頭部模組是可以在繪製角色插圖時，作為參考的頭部模組。可以透過調整輪廓、眼睛、鼻子、嘴巴等部位，建立出自己想要描繪的臉孔。收錄在[素材]面板的[3D]→[頭部]中。

3D人物素材
3D人物素材是可以變更姿勢和表情等的人物素材。收錄在[素材]面板中的[3D]→[人物]中。

3D物件素材
3D物件素材是可以調整角度和材質（質感和色調）等的素材。收錄在[素材]面板的[3D]→[小物品]和[背景]中。
若素材中包含多個元件，可以針對每個元件的位置和尺寸等進行調整。

[素材]面板的[素材[背景]]中也收錄了『3D背景素材』。3D物件素材的素材名稱後方會有「Ver. 2」的記載。
其他辨識方法請參照『3D物件素材和3D背景素材的辨識方法』。

3D背景素材是可以調整角度、材質（質感和色調）

### 選擇攝影機角度

選擇攝影機角度選擇攝影機角度
在CLIP STUDIO PAINT的3D圖層中，可以用各種位置和角度顯示3D素材。完稿時用來調整構圖非常方便。
一旦改變3D圖層的攝影機角度，建立在3D圖層上的透視尺規也會配合攝影機角度同時改變。

備忘

使用滑鼠進行攝影機操作可以在[輔助工具詳細]面板的『滑鼠・手勢』類別中進行變更。

使用移動操縱器，可以改變攝影機角度。關於使用方法，請一併參照『移動操縱器』。
攝影機的旋轉
可以拖拽畫布，將3D圖層的攝影機進行上下左右的旋轉。

平行移動攝影機
可以拖拽畫布，將3D圖層的攝影機進行上下左右的平行移動。以滑鼠操作時，按住滑鼠滾輪並拖拽，也可以進行相同的操作。

攝影機的前後移動
可以拖拽畫布，將3D圖層的攝影機進行前後移動。以滑鼠操作時，按住滑鼠右鍵並拖拽，也可以進行相同的操作。

### 配置3D素材

配置3D素材配置3D素材
可將讀取到CLIP STUDIO PAINT畫布上的3D素材放置在3D圖層的任意位置。
使用移動操縱器
可以使用移動操縱器移動3D素材。選擇3D物件素材的元件時，可以只移動元件。關於使用方法，請一併參照『移動操縱器』。
另外，若選擇了多個3D素材，則無法使用移動操縱器進行移動。
平面移動
可以在畫布的正面將選擇的3D素材上下左右移動。

吸附移動
選擇的3D素材可以吸附在3D空間的地面，或者附近的3D素材上移動。

使用軌道操縱器
沿著軌道操縱器的箭頭進行拖拽，可以將3D素材移動到箭頭的方向。
使用選擇3D物件素材元件後顯示的元件操縱器，也能以相同的操作方式移動元件。

①前後移動沿著藍色箭頭拖拽，3D素材會朝著箭頭的方向移動。
②左右移動沿著紅色箭頭拖拽，3D素材會朝著箭頭的方向移動。
③上下移動沿著綠色箭頭拖拽，3D素材會朝著箭頭的方向移動。

備忘

若將[移動基準軸]設定為[與地面一致]，則軌道操縱器的箭頭將與[輔助工具詳細]面板中的『配置』類別的[位置]相連動。左右移動對應於[X]，上下移動對應於[Y]，前後移動對應於[Z]。

### 旋轉3D素材以改變角度

旋轉3D素材以改變角度旋轉3D素材以改變角度
可將讀取到CLIP STUDIO PAINT畫布上的3D素材，在3D圖層上任意旋轉、改變角度。
使用移動操縱器
使用移動操縱器進行旋轉時的基準位置如下所示。
·3D素描人偶和3D人物素材是在腰部的位置。
·3D物件素材和3D基礎圖形是在軌道操縱器的控制器的位置。
·若只想旋轉3D物件素材的特定元件，則是在選擇元件時顯示的軌道操縱器的控制器之位置。
攝影機視角旋轉
可以將選擇的3D素材或元件旋轉到攝影機的正面。拖拽的方向決定了旋轉的方向。
·橫向拖拽時，以Y軸為基準進行旋轉。
·縱向拖拽時，以X軸為基準進行旋轉。
·其餘方向的拖拽會使其以任意方向旋轉。

平面旋轉
可以將選擇的3D素材或元件旋轉到攝影機的正面。

3D空間基準旋轉
選擇的3D素材和元件可以在3D空間中進行橫向旋轉。

使用軌道操縱器
拖拽軌道操縱器周圍的圓環，可以沿著圓環的方向旋轉3D素材。
使用選擇3D物件素材元件後顯示的元件操縱器，也可以進行相同的操作方式移動元件
使用軌道操縱器進行旋轉時的基準位置，是軌道操縱器的控制器。

### 變更3D素材的縮放比例

變更3D素材的縮放比例變更3D素材的縮放比例
拖拽軌道操縱器和元件操縱器周圍的灰色圓環，可以變更3D素材的縮放比例。能保持3D素材的縱、橫和深度比例，進行放大或縮小。

在[工具屬性]面板和[輔助工具詳細]面板的『配置』類別中，將[固定比例]設定為開啟，即可變更[物件縮放比例]的數值，改變3D素材的縮放比例。
變更3D素材的縮放比例
若為3D物件素材和3D基礎圖形時，軌道操縱器的箭頭前方會顯示方形。此為軸獨立縮放操縱器。
拖拽軸獨立縮放操縱器後，可以個別調整縱、橫和深度。

在[工具屬性]面板和[輔助工具詳細]面板的『配置』類別中，將[固定比例]關閉，即可變更[物件縮放比例]的數值，改變3D素材的縮放比例。 

備忘

即使從已設定親子關係的3D素材中選擇主物件的3D素材，操作軸獨立縮放操縱器，也不會改變子關係3D素材的縮放比例。關於親子關係，請參照『設定親子關係【PRO／EX】』。

恢復變更的縮放比例
『物件啟動器』點擊[重置縮放比例]，可以回到原來的縮放比例。
此外，也可以在[工具屬性]面板或[輔助工具詳細]面板的『配置』類別中點擊[物件縮放比例]的[重置]，將其恢復為原始比例。

### 3D素材操作時的方便功能

3D素材操作時的方便功能3D素材操作時的方便功能
介紹在CLIP STUDIO PAINT中進行3D素材的移動、旋轉和縮放等操作時的便捷功能。
對齊3D模組
開啟『移動操縱器』的[對齊3D模組]後，即可在使用移動操縱器、旋轉操縱器、元件操縱器進行3D素材的移動、旋轉、物件縮放時，顯示邊界方框等效果，並對齊到其他3D素材上。

備忘

使用[工具屬性]面板和[輔助工具詳細]面板進行移動、旋轉和物件縮放的變更時，無法使用[對齊3D模組]。另外，在使用移動操縱器的[吸附移動]移動3D素材的情況下，也無法使用。

在[輔助工具詳細]面板的『操作』類別中的[對齊3D模組]，也可以切換3D模組的對齊。並且可以在[對齊對象]中，從[移動]、[旋轉]和[縮放]選擇欲對齊的對象。此處可以選擇多個項目。
移動
移動3D素材時，可以將其對齊到其他3D素材的位置。進行對齊時，會顯示邊界方框等對齊效果。

旋轉
旋轉3D素材時，會顯示以旋轉中心為基準位置的對齊效果，能以5度為間隔旋轉。可以從[輔助工具詳細]面板的『操作』類別的[旋轉段數角度]中變更對齊角度。

縮放

### 調整3D空間

調整3D空間調整3D空間
可以調整CLIP STUDIO PAINT中3D圖層的顯示方式。使用[霧氣]在3D空間中營造出遠近感，並使用[修正為垂直]調整透視的歪斜。

重要

在CLIP STUDIO PAINT Ver. 1以下版本中，無法編輯設定了[霧氣]和[修正為垂直]的3D圖層。
在[環境設定]對話方塊的『檔案』的[版本相容資訊的保存]中，可以設定在Ver. 1以下版本中顯示的方法。

霧氣
在[輔助工具詳細]面板的[霧氣]類別中開啟[將霧氣設為有效]後，會在3D圖層套用霧氣效果，增加景深。能呈現空氣遠近法般的效果，讓輪廓隨攝影機距離變遠而模糊。

[霧氣的種類]的[距離霧氣]可以根據攝影機與物體的距離而呈現出球狀的濃度。[深度霧氣]可以根據畫面距離平面式增加濃霧效果。
調整[開始距離]與[深度]，則能調整霧氣的效果程度。使用[不透明度]可以調整霧氣的顏色濃度。
[開始距離]可以由自攝影機指定的距離位置開始套用霧氣效果。[深度]為霧氣效果最濃的位置，能指定自[開始距離]的距離。點擊[顏色]，即可從『顏色設定對話方塊』選擇霧氣的顏色。
