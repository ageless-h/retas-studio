# 漫畫・條漫
此章將介紹使用CLIP STUDIO PAINT繪製漫畫的方便功能。除了漫畫分格、對白框與網點之外，還能使用簡單的設定來繪製效果線，以及將圖像輸出為適用於條漫的格式。
·​分格【PRO／EX】
·​對白框【PRO／EX】
·​網點【PRO／EX】
·​流線・集中線【PRO／EX】
·​Webtoon條漫
- 漫畫・條漫

在分格邊框資料夾中，也會建立圖層蒙版。在分格邊框資料夾內的圖層上描畫，可以在不超出分格邊框資料夾的狀態下進行描畫。詳情請參照『在分格邊框中進行描繪』。

### 對白框【PRO／EX】

對白框【PRO／EX】對白框【PRO／EX】
介紹以CLIP STUDIO PAINT的[對白框]工具如描繪圖形般建立對話框的方法。同時也介紹如何編輯已描畫的對白框的方法。
製作對白框   
在[工具]面板上選擇[漫畫]工具。使用[輔助工具]面板中的[對白框]工具，可以像描繪圖形般製作對白框。
另外還有[對白框畫筆]輔助工具，能以像使用筆刷繪製般建立對白框，也有能夠追加對白框尾巴的輔助工具。

描繪對白框
在[工具]面板上選擇[漫畫]工具。從[輔助工具]面板中選擇[對白框]群組，此處選擇[橢圓對白框]輔助工具。

在[工具屬性]面板上設定對白框線條的粗細等。

拖拽畫布後，繪製對白框的同時也會一併在[圖層]面板上建立對白框圖層。

如果已經輸入了文字，繪製對白框時請疊在文字之上，這樣對白框將根據文字的位置進行繪製。同時，文字圖層也會轉換為對白框圖層。另外，於選擇對白框圖層的狀況下繪製對白框時，對白框會被繪製在所選的對白框圖層上。
這些設定可以在[工具屬性]面板和[輔助工具詳細]面板中進行更改。詳情請參照『選擇對白框群組時的輔助工具詳細面板』。

### 流線・集中線【PRO／EX】

流線・集中線【PRO／EX】流線・集中線【PRO／EX】
介紹如何使用CLIP STUDIO PAINT[圖形]工具中的[流線]群組和[集中線]群組的輔助工具，輕鬆繪製效果線的方法。用此方法繪製出的效果線，在完成後還能更改線的方向和數量等。
製作流線
流線群組
在[工具]面板上選擇[漫畫]工具。在[輔助工具]面板上，從[流線]中選擇一個輔助工具。

點擊畫布，繪製曲線。雙擊後，曲線將被確定。也可以點擊畫布以外的範圍來確定曲線。在畫布上繪製流線時，將同時在[圖層]面板上建立流線圖層。

使用[工具屬性]面板和[輔助工具詳細]面板，除了在此說明的內容之外，還可以進行各種設定。可進行流線的追加方法和線的間隔等設定。
[流線]群組的[輔助工具詳細]面板中，可以設定『流線【PRO／EX】』・『連續曲線』・『描繪間隔【PRO／EX】』・『描繪位置【PRO／EX】』的各項目。
除此之外的設定與筆刷相同。詳情請參照『筆刷的設定項目』。
使用特殊尺規
在畫布上建立平行線尺規後，可以將沿著尺規的平行線繪製到點陣圖層或向量圖層上。詳情請參照『平行線』。

### 網點【PRO／EX】

網點【PRO／EX】網點【PRO／EX】

建立網點化的圖層
此處將以半色調的網點為例，介紹貼網點的方法。在想要貼上網點的位置建立選擇範圍。關於建立選擇範圍的方法，請參照『選擇範圍』。
可以使用以下任一方法來建立網點化的圖層。
·選擇[圖層]選單→[新圖層]→[網點]。
·從選擇範圍啟動器中點擊[新網點]。

顯示[簡易網點設定]對話方塊後，設定網點的線數和濃度，最後點選[OK]。
即會在選擇範圍中貼上透過[簡易網點設定]對話方塊所設定的網點，並在[圖層]面板中建立網點化的平塗圖層和圖層蒙版。

備忘

### Webtoon條漫

Webtoon條漫Webtoon條漫

Webtoon顯示框
這是在智慧型手機等設備上檢視Webtoon時的參考框架。在畫布上顯示Webtoon顯示框後，框外會以灰色顯示。或者，可以用視窗的顏色來填充灰色部分以進行隱藏。

備忘

顯示Webtoon顯示框
選擇[檢視]選單→[Webtoon顯示框]，可以切換Webtoon顯示框的顯示或隱藏。
無論設定如何，以下情況都無法顯示Webtoon顯示框。

開啟[填充Webtoon顯示範圍外的區域]功能後，Webtoon顯示範圍外將以視窗的顏色進行填滿並隱藏。
