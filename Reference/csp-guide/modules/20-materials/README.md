### 讀取素材的方法

讀取素材的方法讀取素材的方法
從CLIP STUDIO ASSETS下載的素材或收錄在[素材]面板上的素材，可以透過點擊[素材]面板的[貼上素材]讀取至畫布或面板中。

依照素材種類的不同，會讀取至不同的地方，如：畫布、[輔助工具]面板。關於各種類的詳細步驟，請參照以下內容。
·『圖像素材』
·『範本素材』
·『筆刷素材』
·『輔助工具素材』
·『色板素材』
·『3D素材』
·『自動動作素材【PRO／EX】』
·『漸層組素材【PRO／EX】』
·『工作區素材【PRO／EX】』
·『動畫素材【PRO／EX】』
無法直接讀取的素材
以下素材無法讀取。但如果進行下載，有可能可以讀取。
·素材的右上方顯示有箭頭的雲朵圖示，表示該素材尚未下載完成。選擇素材並執行[貼上素材]後，會顯示確認訊息。點擊[是]，會啟動CLIP STUDIO並開始下載素材。下載後，可以像平常一樣使用素材。
·素材的右上方有顯示禁止標誌圖示時的素材，代表無法在您所使用的CLIP STUDIO PAINT上使用。點擊禁止標誌圖示，會顯示無法在CLIP STUDIO PAINT中使用的原因。

### 檢索素材

檢索素材檢索素材
在『素材面板』或『追加素材對話方塊』中，可以使用檢索欄和標籤清單來搜索素材。可以結合使用檢索欄和標籤清單進行檢索。另外，加入[收藏]可以更容易找到。

從檢索欄進行檢索
在檢索欄中輸入關鍵字，可以從追加素材對話方塊中顯示的素材，或從在[素材]面板中選擇的資料夾中進行搜索。
·關鍵字的檢索範圍是素材名稱與標籤所使用的字串。部分相符的單詞也可以進行檢索。
·單字前後加上「"」(雙引號)，可以檢索包含空格的單詞。
·平假名、片假名可以一起檢索，不需要區分。
·於多個關鍵字間輸入空格，即可進行限定範圍檢索。增加關鍵字可以進一步篩選素材。
點擊右側的[×]刪除關鍵字，即可解除搜尋條件。
從標籤清單中進行檢索
點擊標籤清單中的標籤按鈕，可以只顯示設定相同標籤的素材。再次點擊相同標籤的按鈕，可以解除搜尋。
點擊多個標籤可以進行檢索篩選。

備忘

·標籤清單中的標籤可以分為以下種類。[類別]是根據素材的獲取方式自動添加的標籤。[預設標籤]是根據素材的種類和數據自動添加的標籤。[用戶標籤]是附加於素材本身的標籤。
·設定『素材屬性對話方塊【PRO／EX】』，可以將標籤添加到素材中。


### 素材面板

素材面板素材面板
介紹管理CLIP STUDIO PAINT素材的[素材]面板功能。可以切換素材的顯示方式，並使用資料夾來整理素材。

A.在ASSETS上尋找素材
點選後，將啟動CLIP STUDIO並顯示CLIP STUDIO ASSETS。可以將在CLIP STUDIO ASSETS上公開的素材下載到CLIP STUDIO PAINT中。

備忘

從CLIP STUDIO ASSETS下載的素材，可以點擊樹狀顯示中的[下載]資料夾來查看。

B.樹狀顯示
[素材]面板的素材是按照種類以資料夾進行管理。點選資料夾，即可在素材一覽中顯示資料夾內的素材。點擊兩下資料夾或資料夾名稱，可以打開下一層的資料夾。
C.編輯素材資料夾的命令列
可以追加和刪除樹狀顯示的素材資料夾。
①建立素材資料夾可以在樹狀顯示中新增資料夾。
②刪除素材資料夾可以在樹狀顯示中刪除選擇中的資料夾。另外，無法刪除初始設定的資料夾。

重要

刪除資料夾時，資料夾中的素材也會被刪除。

③變更素材資料夾名稱可以在樹狀顯示中變更選擇中資料夾的名稱。另外，無法變更初始設定的資料夾。
D.檢索欄
請參照『從檢索欄進行檢索』

關於CLIP STUDIO ASSETS的使用方法和素材使用規則，請參照CLIP STUDIO ASSETS的說明。

### 自製的素材登記【PRO／EX】

自製的素材登記【PRO／EX】自製的素材登記【PRO／EX】
將常用的圖像和圖層結構等作為素材登記，可以方便重複使用。另外，可以將登記在[素材]面板中的自製素材備份到CLIP STUDIO雲端。在其他環境中即可使用相同的素材。
將圖層登記為素材
可以將畫布上繪製的內容登記為素材。從[圖層]面板中選擇想要作為素材登記的圖層。依據所選擇的圖層，素材的種類將會隨之改變。
·若只選擇一張點陣圖層，則可以將其登記為圖像素材圖層。
·若選擇了多個圖層，則可將其登記為與原圖層相同種類的圖像素材。
·若選擇了向量圖層，則可保持向量圖層的狀態將其登記為圖像素材。
·若選擇了對白框圖層，則可將其登記為對白框素材。
·若選擇了圖像素材圖層，則可將其登記為圖像素材圖層。
在畫布中想要作為素材的部份上建立選擇範圍。如果不建立選擇範圍，將以整個圖層的描繪範圍為對象。

執行以下任一操作。
·選擇[編輯]選單→[登記素材]→[圖像]。
·從[素材]面板的選單中選擇[將圖像登記為素材]。

當『素材屬性對話方塊【PRO／EX】』顯示後，設定素材的資訊和保存位置。

### 物品庫面板

物品庫面板物品庫面板
[物品庫]面板可以將檔案和素材登記至物品清單中。已登記的檔案或素材，可以像一般的素材一樣，將其拖拽至畫布上，即可讀取至畫布中。當想在畫布內使用多個相同的檔案或素材時，非常方便。

備忘

[物品庫]中登記的檔案和素材會與畫布連結。其他畫布無法使用。

①項目清單[物品庫]面板中顯示了已登記的檔案和素材一覽。關於可以登記在[物品庫]面板中的素材和檔案，請參照『可以登記至物品庫面板上的素材』。

②登記會顯示打開檔案的對話方塊，可將檔案登記到[物品庫]面板中。於Windows／macOS／iPad環境中，可以將檔案拖拽至項目清單中，以便登記檔案。
③設定可以設定從項目清單中選擇的項目名稱。顯示[物品設定]對話方塊後，在[物品名稱]中輸入名稱，點選[OK]。所輸入的名稱將會套用至項目清單。
若為音頻檔案時，可以點擊[物品設定]對話方塊中的[播放]按鈕來播放檔案。
④刪除可以從項目清單中刪除所選擇的項目。然而，無法刪除已在畫布上使用的素材和檔案。請從[圖層]面板中刪除。
