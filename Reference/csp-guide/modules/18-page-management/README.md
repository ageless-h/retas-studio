### 寫出多頁作品

寫出多頁作品寫出多頁作品
使用CLIP STUDIO PAIN製作的多頁作品可以一口氣寫出為通用檔案，也可以匯出PDF、EPUB等格式。
一併寫出

可以書出的檔案格式包括BMP・JPEG・PNG・WebP・TIFF・Targa・Adobe Photoshop文件（副檔名：psd）・Adobe Photoshop大型文件（副檔名：psb）・PDF格式（副檔名：pdf）。

3顯示[寫出設定]對話方塊後，對於要寫出的檔案進行設定。
關於設定項目，請參照『寫出設定對話方塊』。
另外，如果透過[一併寫出]對話方塊顯示[寫出設定]對話方塊，不會顯示[輸出時預覽渲染結果]。另外，無法從[輸出範圍]中選擇[選擇範圍]。

### 關於管理檔・頁面檔案

關於管理檔・頁面檔案關於管理檔・頁面檔案
若新建以多個頁面構成的作品並進行編輯，會建立用來管理作品整體的管理資料夾和管理檔（副檔名：cmc）。
管理資料夾內存放著每個頁面的CLIP STUDIO FORMAT（副檔名：clip）檔案，並透過管理檔來進行每個頁面的管理。

重要

請勿直接在操作系統的檔案總管或Finder等工具中，替換多頁作品管理資料夾內的頁面管理檔或各個頁面的檔案。檔案可能會損壞，無法正常開啟。請使用頁面管理視窗來進行作品的管理。詳情請參照『頁面管理視窗的操作』。

建立管理檔
若想建立多頁作品，在[新建]對話方塊中設定[多頁]，即可進行新建。詳情請參照『新建立畫布』。
此外，在編輯單頁畫布時，進行以下操作，將顯示[建立頁面管理資料夾]對話方塊，可以建立作品資料夾和管理檔。

·『複製頁面』

·『顯示前後的頁面』

### 頁面管理視窗的操作

頁面管理視窗的操作頁面管理視窗的操作
CLIP STUDIO PAINT的頁面管理視窗是用來管理多個頁面的畫面。
從[檔案]選單→[打開]開啟管理檔（副檔名：cmc），或者在打開作品中任意頁面的狀態下，選擇[頁面管理]選單→[打開頁面管理視窗]，即可顯示頁面管理視窗。
頁面管理視窗的顯示，會根據作品設定而有所不同。每個頁面的縮圖會排列在一起，能像書本一樣閱讀的作品會以橫向顯示，Webtoon作品則以縱向方式呈現。

透過頁面管理視窗，可以選擇頁面並進行編輯。點擊頁面的縮圖可選擇頁面。若想選擇多個頁面，請按住[Ctrl]鍵並點擊頁面。若想連續選擇多個頁面，請在按住[Shift]鍵的同時，點擊要選擇的第1個和最後1個頁面。
移動頁面
從頁面管理視窗中選擇要移動的頁面，並將其拖拽到想要移動到的位置上。

頁面的排序方法
如果無法拖拽頁面，請檢查[頁面管理]選單→[頁面的排序方法]的設定。若為[Ctrl+拖拽]，請一邊按住[Ctrl]鍵，一邊拖拽頁面。
要使頁面能夠透過拖拽移動，請選擇[頁面管理]選單→[頁面排序方式]，然後選擇[拖拽]。
設為[Ctrl+拖拽]，可以避免錯誤操作導致頁面互換等

### 使用情節編輯器

使用情節編輯器使用情節編輯器
CLIP STUDIO PAINT的情節編輯器是可以一併輸入和編輯多頁文字的編輯器。選擇[頁面管理]選單→[編輯文字]→[情節編輯器]，即可顯示。

重要

·顯示情節編輯器的同時，所有頁面的編輯歷史記錄將被刪除。[歷史記錄]面板或[撤銷]等操作將無法執行。在顯示情節編輯器後，情節編輯器的編輯歷史將被記錄在[歷史記錄]面板中，但當關閉情節編輯器後，所有記錄將被刪除。
·CLIP STUDIO PAINT Ver. 1.4.1之前版本，以及在ComicStudio和IllustStudio中輸入的文字，在以情節編輯器進行編輯之前，需要將文字替換為最新的描繪。選擇[文字]工具或[物件]輔助工具時，會顯示確認訊息。點擊[套用於圖層內的所有文字]後，將會替換為最新的描繪。

情節編輯器中包含可以一覽顯示頁面內文字的「頁面區域」，和按照對話逐句顯示文字「文字區域」。作品內如果有已輸入的文字，文字區域會呈現顯示已輸入的文字的狀態。但如果在作品中將對白框圖層或文字圖層設為隱藏，將無法在情節編輯器中進行編輯。

### 協同作業【Windows／macOS】

協同作業【Windows／macOS】協同作業【Windows／macOS】
針對多頁作品，使用此功能可以讓多個用戶進行協同作業。
若想進行協同作業，須由用戶將作品資料夾作為「協同作業數據」上傳至網路伺服器或共享資料夾上。作品資料夾的上傳必須由用戶執行，其權限為「管理員」，可以指定作業負責人。
進行作業的用戶需取得協同作業數據，並在各自的裝置上建立「作業資料夾」。進行下載的用戶的權限為「使用者」。由管理員指定編輯頁面的用戶將成為「作業負責人」。
各使用者可以編輯作業資料夾內的頁面，並將編輯內容反映至網路上的協同作業數據。

重要

在使用Windows和macOS之間的協同作業功能時，請將協同作業數據上傳到可以在網路上共享的硬碟或伺服器上。另外，以下環境無法使用。
·如果在Windows裝置上保存的協同作業數據已設定檔案共享，將無法以Mac OS X 10.9之前的環境存取。
·如果在macOS裝置上保存的協同作業數據已設定檔案共享，將無法以Windows環境存取。

開啟想要進行協同作業的頁面管理檔，選擇[頁面管理]選單→[協同作業]→[準備協同作業數據]。
