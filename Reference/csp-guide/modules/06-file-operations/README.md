### 縮時攝影

縮時攝影縮時攝影
可以將在CLIP STUDIO PAINT畫布上進行的作業內容記錄為縮時攝影影片。
記錄縮時攝影
在[新建]對話方塊中設定[記錄縮時攝影]，即可開始記錄縮時攝影。

·選擇[品質優先]，會以品質為優先的狀態記錄縮時攝影。由於縮時攝影的記錄內容和設定狀態會保存在CLIP STUDIO FORMAT（副檔名：clip）的檔案中，因此會造成作品的檔案較大。
·選擇[檔案尺寸優先]，會以縮小檔案尺寸為優先的狀態記錄縮時攝影。因此縮時攝影的品質會較低。
·選擇[不記錄]，則不會記錄縮時攝影。
[記錄縮時攝影]的記錄內容和設定狀態會保存在CLIP STUDIO FORMAT（副檔名：clip）的檔案中。保存作品，並再次以CLIP STUDIO PAINT開啟檔案進行作業，就會自動繼續記錄縮時攝影。
如想從途中開始記錄縮時攝影時，請參閱下方的『從途中開始記錄縮時攝影』。

備忘

關於[新建]對話方塊，請參照『新建立畫布』。

從途中開始記錄縮時攝影
若想在製作途中開始記錄縮時攝影，請在以下的對話方況設定[記錄縮時攝影]、[品質優先]或是[檔案尺寸優先]。

### 保存檔案

保存檔案保存檔案
可以將在CLIP STUDIO PAINT中編輯的畫布保存到檔案中。保存的支援格式包括CLIP STUDIO FORMAT（副檔名：clip）、CLIP STUDIO PAINT格式（副檔名：lip）、BMP、JPEG、PNG、WebP、TIFF、Targa、Adobe Photoshop文件（副檔名：psd）和Adobe Photoshop大型文件（副檔名：psb）。
·如果想保留圖層結構來保存，請以CLIP STUDIO PAINT FORMAT（副檔名：clip）進行保存。
·如果想維持圖層結構來使用於其他應用程式，請使用『保存複製』，保存為Adobe Photoshop文件或Adobe Photoshop大型文件。
·若要將圖層合併並以泛用格式保存，以便使用於網頁等，建議利用[平面化影像並寫出]或[寫出至照片圖庫]。詳情請參照『寫出檔案』。

重要

從CLIP STUDIO打開的ibisPaint作品檔案（副檔名：ipv）和「kakooyo！」的作品無法以相同的檔案格式進行保存。新的檔案將以CLIP STUDIO FORMAT（副檔名：clip）的格式保存。

### 顯示預覽色彩設定檔【PRO／EX】

在CLIP STUDIO PAINT中，可以選擇色彩設定檔，並在畫布上進行預覽顯示。

[直覺]會保護顏色之間的視覺關係，使顏色在彩色數值更改的情況下仍然顯得自然。[彩度]會將原始色彩空間的最大亮度與轉換目標色彩空間的最大亮度進行比較

### 變更畫布尺寸

變更畫布尺寸變更畫布尺寸
以CLIP STUDIO PAINT建立畫布後，有多種方法可以更改尺寸。包括『變更圖像解析度』、『變更畫布尺寸』、『變更畫布高度』、『使畫布尺寸符合選擇範圍』、『旋轉・反轉畫布』等。

重要

DEBUT的畫布尺寸最大可達10000px。無法設定超過此尺寸的大小。

變更圖像解析度
