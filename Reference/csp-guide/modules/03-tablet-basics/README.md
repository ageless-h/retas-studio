### 簡約模式和工作室模式

簡約模式和工作室模式簡約模式和工作室模式
平板電腦版有簡約模式和工作室模式。可以依照繪製時的操作性和用途使用不同的介面。

簡約模式
嚴選繪圖功能，適合以平板電腦繪圖的介面。將工具等功能的配置精簡化，擴展畫布的可畫空間。
關於簡約模式的使用方法請參照『簡約模式的介面』和『簡約模式的基本操作』。
工作室模式
此模式下可以使用CLIP STUDIO PAINT的所有功能。可以自訂操作介面，在自己喜歡的環境中進行描繪。原則上，本手冊講解的是工作室模式的使用方法。
關於工作室模式的使用方式請參照『工作室模式的介面』和『工作室模式的基本操作』。

在創作作品的過程中可以隨時切換簡約模式和工作室模式的介面。
可以依照用途自由切換，像是：在使用簡約模式時暫時切換到工作室模式使用功能，或者在使用工作室模式時切換到簡約模式以更大範圍地使用畫布。
關於切換方法，請參照『切換為簡約模式』和『切換為工作室模式』。

### 以手指和筆使用不同工具

以手指和筆使用不同工具以手指和筆使用不同工具
若在平板電腦版CLIP STUDIO PAINT上使用專用的感應筆進行描繪，可以設定讓手指和感應筆執行不同的操作。當手指觸碰到畫布時，可以防止因誤觸產生的不必要的描繪等。簡約模式與工作室模式的設定方法不同。
簡約模式
從[選單]中選擇[環境設定和說明]。

關閉[以手指描繪]後，將無法使用手指在畫布上進行描繪。
關閉後，仍可使用手指操作畫布。關於畫布的操作方法，請參照『觸控手勢』。

工作室模式
點擊開啟命令列上的[以手指和筆使用不同工具]，即可使用手指和感應筆進行不同的操作。

開啟[以手指和筆使用不同工具]後，於初始狀態設定中可進行以下操作。
·移動畫布（[手掌]工具）：單指滑動
·取得顏色([吸管]工具)：單指長押

簡約模式的介面

①CLIP STUDIO能啟動CLIP STUDIO。CLIP STUDIO是啟動CLIP STUDIO PAINT時顯示的畫面。要返回原來的畫面，請在顯示CLIP STUDIO後點擊[動手描繪]。
②保存可以保存在CLIP STUDIO PAINT的畫布上繪製的作品。
③撤銷點擊後，可以撤銷前一個操作。
④重做點擊後，可以重新執行已取消的操作。
⑤素材點擊後會顯示素材的清單。選擇素材的種類，點擊想要使用的素材，素材就會顯示在畫布上。點擊畫布右上方的[OK]，可以確定讀取素材。點擊畫布左上方的[取消]，即可取消素材的讀取。
⑥選單點選後，會顯示選單。可以進行各種操作，例如更改畫布尺寸，自訂介面顏色和方向等。
⑦工具列可在CLIP STUDIO PAINT使用的工具清單。點擊圖示即可選擇工具。再次點擊相同工具的圖示，即可進行更詳細的設定。關於描繪

備忘

選擇描繪的工具
從工具列中點擊毛筆的圖示，即可使用毛筆進行描繪。

再次點擊毛筆的圖示，即可選擇初始收錄的筆刷。從左側選擇類別後，將會顯示所選類別中包含的筆刷一覽。
從筆刷一覽中，點選想要使用的筆刷。

開始在畫布上進行描繪後，筆刷一覽將被隱藏。筆刷尺寸可以在畫布下方的筆刷尺寸滑桿上進行變更。

選擇顏色
選擇要用筆刷繪製的顏色時，請點擊工具列下方的顏色切換。由上至下依序為主顏色、輔助顏色和透明色。

①主顏色點擊圖示後，可以使用主顏色進行筆刷的描繪或填充。再次點擊即可顯示色彩面板，並可變更顏色。
