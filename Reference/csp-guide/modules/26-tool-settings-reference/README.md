環境
若使用[物件]輔助工具選取3D圖層或3D素材，可以設定3D圖層的顯示方式等。

①渲染設定點擊後會顯示『渲染設定對話方塊』，可以進行3D素材的輪廓線和照明（光源）等設定。

備忘

·關於CLIP STUDIO MODELER，請參照創作支援網站CLIP STUDIO。
·讀取已進行物理設定的3D人物素材時，CLIP STUDIO PAINT的操作可能變得緩慢。將[實體模擬]關閉可能會有所改善。

③物理設定點選後，會顯示『物理設定對話方塊』，可以設定3D人物素材的物理演算套用方式。
④設定編輯顯示可以從[普通]和[高速]中選擇3D圖層的編輯時顯示方式。

④均勻配置設為開啟，排列為圓形的文字、單詞的間隔，將在起點和終點之間均勻配置。關於起點和終點請參照『沿著圓形輸入文字』 。

顏色變化
可以改變所選擇的描繪色並進行描繪。可以根據筆壓和筆劃的速度等，調整描繪色的色相、彩度和明度，或者使用隨機的顏色進行描繪。

①筆刷前端的變化開啟後，可以根據描繪顏色的[色相]、[彩度]、[明度]、[輔助描繪色混合率]使顏色變化並進行描繪。

[輔助描繪色混合率]可以用混合了主描繪色和輔助描繪色的顏色來進行描繪。數值越大，輔助描繪色的混合比例越大。點擊各別的[影響源]，可以透過筆壓和筆劃的速度套用顏色的變化。關於[影響源]，請參照『影響源的設定』 。
②依筆劃隨機變化開啟後，顏色會以描繪顏色的[色相]、[彩度]、[明度]和[輔助描繪色混合率]為基準，在繪製的各別條線上發生變化。各項目的數值增加，顏色的變化也會增加。
③變化對象若使用同時具有主描繪色與輔助描繪色的筆刷前端形狀，或者設定了[輔助描繪色混合率]，則可以在[筆刷前端的變化]和[依筆劃隨機變化]的[色相]・[彩度]・[明度]之中，從[主描繪色與輔助描繪色]・[主描繪色]・[輔助描繪色]來選擇的套用對象。
