### 自動選擇工具

自動選擇工具自動選擇工具
CLIP STUDIO PAINT的[工具]面板中的[自動選擇]工具，當點擊畫布時，可以在點擊位置的相同顏色部分或被線條圈住的部分建立選擇範圍。

### 快速蒙版【PRO／EX】

快速蒙版【PRO／EX】快速蒙版【PRO／EX】
CLIP STUDIO PAINT的快速蒙版，可以在查看圖像的同時，建立形狀複雜的選擇範圍，是相當便利的功能。
選擇[選擇範圍]選單→[快速蒙版]，在[圖層]面板中建立快速蒙版。選擇快速蒙版，使用[沾水筆]工具或[填充]工具等進行描繪。
再次選擇[選擇範圍]選單→[快速蒙版]，將在快速蒙版上繪製的內容轉換為選擇範圍。進行選擇範圍轉換的同時，快速蒙版將被刪除。

備忘

·在建立選擇範圍後，選擇[選擇範圍]選單→[快速蒙版]，選擇範圍將反映到快速蒙版上。
·快速蒙版的顏色可以在[圖層屬性]面板中進行更改。詳情請參照『圖層顏色』。
·快速蒙版無法保存。保存時將同步變換為選擇範圍。如果想要中斷操作或將選擇範圍保存到檔案中，使用選擇範圍圖層會很方便。詳情請參照『將選擇範圍設定為圖層【PRO／EX】』。

### 將選擇範圍設定為圖層【PRO／EX】

將選擇範圍設定為圖層【PRO／EX】將選擇範圍設定為圖層【PRO／EX】
CLIP STUDIO PAINT的選擇範圍圖層，是能用來保存已建立的選擇範圍的圖層。保存選擇範圍，即可重複使用相同的選擇範圍。
此外，如果以CLIP STUDIOFORMAT（副檔名：clip）保存檔案，保存後仍然可以繼續使用選擇範圍圖層。若以其他格式保存，將在保存的同時進行刪除。此外，選擇範圍圖層無法進行列印。
儲存選擇範圍
選擇[選擇範圍]選單→[儲存選擇範圍]，即可在[圖層]面板中建立選擇範圍圖層。
選擇範圍建立後，執行[儲存選擇範圍]，將建立選擇範圍圖層，並使選擇範圍反映至選擇範圍圖層中。
在選擇範圍圖層中使用[沾水筆]工具或[填充]工具進行描畫，可以編輯選擇範圍。

備忘

選擇範圍圖層的顏色可以在[圖層屬性]面板中進行更改。詳情請參照『圖層顏色』。

從儲存中還原選擇範圍
選擇[選擇範圍]選單→[從儲存中還原選擇範圍]，可以利用選擇範圍圖層來建立選擇範圍。
選擇範圍建立後，選擇範圍圖層仍會保留在[圖層]面板中。可以再次建立相同的選擇範圍。
