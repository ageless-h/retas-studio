關於本書的標記
本書（包括數位版）根據2025年4月的產品程式進行撰寫與編輯，因此可能與實際產品程式的規格有所不同。
·本書記載Windows／macOS／iPad／Android的使用方法。除了『夥伴模式』之外，關於iPhone和Android智慧型手機的使用方法請參考CLIP STUDIO TIPS。
·除非特別註明，本書的按鍵標記皆比照Windows記載。請參照『關於操作標記』後，重新解讀操作。
關於符號
本書中使用記號解釋與操作相關的事項。記號的意義如下。
重要
操作CLIP STUDIO PAINT時需要注意的事項。
備忘
記載有關CLIP STUDIO PAINT操作的補充說明等。
關於製品級別
CLIP STUDIO PAINT有DEBUT、PRO、EX三種產品級別，本書將描述所有級別的操作方法。關於各級別的限定功能，皆有在產品級別標示中記載。
各級別所支援的功能，請參照產品列表。
