### 夥伴模式

夥伴模式夥伴模式

重要

連接智慧型手機

### CLIP STUDIO TABMATE 2

CLIP STUDIO TABMATE 2CLIP STUDIO TABMATE 2
使用另外販售的單手輸入裝置CLIP STUDIO TABMATE 2，可以為其設定進行CLIP STUDIO PAINT的各種操作。

備忘

關於CLIP STUDIO TABMATE2，請參照CLIP STUDIO TABMATE的產品概要。

首次連接時，請執行以下的選單命令。
·使用Windows版，請選擇[檔案]選單→[CLIP STUDIO TABMATE]→[登記新的TABMATE 2]。
·使用macOS／平板電腦版，請選擇[CLIP STUDIO PAINT]選單→[CLIP STUDIO TABMATE]→[登記新的TABMATE 2]。

備忘

若顯示請求確認使用藍芽的訊息，請點擊[OK]。

若顯示「正在搜尋新的TABMATE 2…」的訊息，請長按3秒CLIP STUDIO TABMATE 2機身的[CLIP STUDIO]按鈕，以開啟電源。

當顯示「已找到新的TABMATE 2。」訊息時，請點擊[連接]。

### QUMARION【Windows／macOS】

QUMARION【Windows／macOS】QUMARION【Windows／macOS】   
可以連接QUMARION，為CLIP STUDIO PAINT的3D素描人偶或3D人物素材設定姿勢。請參照QUMARION附帶的手冊，了解QUMARION裝置的連接方法和操作方法。

重要

·QUMARION已停止販售。
·請勿在CLIP STUDIO PAINT啟動的情況下為電腦連接或移除QUMARION。CLIP STUDIO PAINT的操作可能會變得不穩定。
·搭載Apple M系列處理器的macOS系統，不支援QUMARION。

使用QUMARION
接上QUMARION後，進行以下操作。
·若為Windows，請選擇[檔案]選單→[QUMARION]→[使用QUMARION]。
·若為macOS，請選擇[CLIP STUDIO PAINT]選單→[QUMARION]→[使用QUMARION]。
在畫布上讀取3D素描人偶和3D人物素材。點擊物件啟動器的[QUMARION有效]，將啟用來自QUMARION的輸入。

### Galaxy系列【Android】

Galaxy系列【Android】Galaxy系列【Android】
若在Samsung Galaxy系列的設備上使用CLIP STUDIO PAINT，可以使用Galaxy系列獨有的功能。
DeX模式
將支援DeX模式的Galaxy系列設備連接到顯示器或WacomOne上，就可以在大螢幕上觀看並操作CLIP STUDIO PAINT。

備忘

關於如何在Galaxy系列中使用DeX模式連接的方法，請參照Galaxy系列的設定畫面或說明。

在外接螢幕上顯示畫布
[檢視]選單→選擇[在外接螢幕上顯示畫布]，連接的顯示器上僅會顯示畫布。在大畫面上確認作品將更佳方便。
在電腦上執行操作
在Windows上安裝DeX模式的應用程式，並連接Galaxy系列的平板電腦或智慧型手機，即可在Windows設備上操作安裝在Galaxy系列上的CLIP STUDIO PAINT。
在DeX模式下進行顯示，將會如下切換CLIP STUDIO PAINT的顯示。
·初始設定狀態下，畫布會顯示捲軸。
·初期設定的狀態下，筆刷類游標和畫筆類游標的游標形狀會是筆刷尺寸。
隔空操作
Galaxy Tab S系列等

### iPad

iPadiPad
若在iPad上利用CLIP STUDIO PAINT，可以使用iPad和Apple pencil獨有的功能。關於詳細設定方法和各個功能的支援環境，請參照Apple公司的產品說明。
·使用Apple Pencil的隨手寫功能，能以手寫方式輸入文字。如選擇[文字]工具在畫布上輸入文字，或在對話方塊和面板中鍵入數值等。
·也能使用Apple Pencil Pro的雙指按壓功能。用力按壓Apple Pencil Pro，會顯示[快速存取]面板。若想使用雙指按壓，必須在iPadOS中進行設定。
Wacom觸控筆【iPad】
使用兼容iPad的Wacom觸控筆可以在CLIP STUDIO PAINT上進行繪畫。關於如何將Wacom觸控筆連接到iPad上，請參考Wacom觸控筆的使用手冊等資料。
使用Wacom觸控筆
選擇[CLIP STUDIO PAINT]選單→[Wacom觸控筆]→[使用Wacom觸控筆]，並將其設定為開啟，這樣就可以在CLIP STUDIO PAINT中使用連接到iPad的Wacom觸控筆了。
Wacom觸控筆設定
選擇[CLIP STUDIO PAINT]

### Surface Dial【Windows】

Surface Dial【Windows】Surface Dial【Windows】
使用Surface Dial可以操作 CLIP STUDIO PAINT。關於Surface Dial的連接方法和基本操作方法，請參照Surface Dial的說明。
從Surface Dial上選擇功能。
啟動CLIP STUDIO PAINT後，請長按Surface Dial。螢幕上會顯示Surface Dial的選單。
旋轉Surface Dial以切換功能。每次切換功能時，Surface Dial都會震動。選擇中的功能名稱將顯示在選單的中心位置。

當想要使用的功能顯示在選單的中央時，請點選Surface Dial。會隱藏選單，並可執行所選擇的功能。機能選擇後的操作，請參照下一個『Surface Dial可以使用的功能』。
另外，若想重新選擇其他功能，請再次長按Surface Dial並顯示選單。
