# 尺規的使用方法【PRO／EX】
介紹CLIP STUDIO PAINT的尺規功能。除了直線和曲線的尺規之外，還有可以繪製集中線和流線的特殊尺規，以及能指定消失點並繪製透視的透視尺規。
·​尺規的種類和建立方法
·​對齊到尺規進行描繪
·​編輯尺規
·​編輯透視尺規
·​在透視尺規上進行描繪
·​刪除尺規
- 尺規的使用方法【PRO／EX】

### 尺規的種類和建立方法

尺規的種類和建立方法尺規的種類和建立方法
CLIP STUDIO PAINT的[尺規]是用來輔助準確繪製直線和曲線的工具。選擇[尺規]工具後，可以透過[輔助工具]面板建立各種尺規。
這裡將以從[輔助工具]面板中選擇[圖形尺規]輔助工具為例進行說明。

請在[工具屬性]面板的[圖形]中選擇要設定為尺規的圖形。開啟[在編輯圖層上建立]選項後，可以在[圖層]面板中選擇的圖層上建立尺規。

在畫布上拖拽，可以繪製圖形。這裡繪製的圖形會成為尺規，可以使用[沾水筆]工具等繪圖工具來描摹繪製。建立尺規後，[圖層]面板的圖層上會顯示尺規的圖示。

尺規的建立方法，會根據尺規的種類而異。
關於已建立的尺規的使用方法，請參照『對齊到尺規進行描繪』。
直線尺規輔助工具
[直線尺規]輔助工具可以建立直線的尺規。在畫布上進行拖拽，即可建立直線尺規。邊按著[Shift]鍵邊進行拖拽，能以45°的間隔描繪直線。

使用[工具屬性]面板和[輔助工具詳細]面板，還可以進行本章節說明內容之外的各種設定。可以讓尺規顯示刻度，或者建立曲線的尺規。
在[直線]輔助工具的[輔助工具詳細]面板中，可以從『建立尺規【PRO／EX】』

### 編輯尺規

以下將介紹在CLIP STUDIO PAINT中，如何調整已建立的尺規的位置和角度，以及變更尺規形狀的方法。
直線尺規・曲線尺規・圖形尺規・尺規筆
針對使用[直線尺規]・[曲線尺規]・[圖形尺規]・[尺規筆]等各個輔助工具所建立的尺規，當選擇尺規或控制點時，可以分別編輯尺規或控制點。
選擇[操作]工具的[物件]輔助工具，點擊畫布中的尺規線條，可以編輯顯示節點和控制點的部分之尺規線條。邊按住[Shift]鍵邊進行點擊，可以選擇複數尺規的線。

若想進一步選擇控制點，請使用[物件]輔助工具點擊想要選擇的控制點。邊按住[Shift]鍵邊進行點擊，可以選擇複數控制點。

尺規的移動和變形
尺規的移動・旋轉・變形方法與編輯向量圖層時相同。詳情請參照『執行向量圖層的變形』。

備忘

進行尺規的移動和變形時，能對齊至其他物件。關於能對齊的物件，請參照『對齊目標的物件』。

### 編輯透視尺規

編輯透視尺規編輯透視尺規
以下將介紹如何編輯CLIP STUDIO PAINT的透視尺規。
選擇[操作]工具中的[物件]輔助工具，再點選畫布中的透視尺規線，會顯示輔助線、消失點和節點，即可進行編輯。

此外，使用[尺規]工具中的[透視尺規]輔助工具，可以變更透視尺規的透視法，和追加輔助線。
透視尺規的各部分名稱和功能
以下將解說選擇[物件]輔助工具時顯示的輔助線和節點。

備忘

透視尺規的節點尺寸，可以在[環境設定]對話方塊中『工具』的[直線和曲線的節點]中進行變更。

①視平線呈現場景中觀察者的眼睛高度，或是攝影機的高度的線。與畫面上的水平線（或地平線）一致。繪製比視平線高的物體會呈現仰視構圖，比視平線低的物體會呈現俯瞰構圖。
②消失點物體的寬度・深度・高度的線條會收斂到一個點。相對於地面，平行的寬度、深度方向的消失點，位置一定在視平線上。依照透視圖不同，消失點的數量也會有所不同。
③輔助線從消失點延伸出的一對線。消失點上一定會有輔助線。除了繪圖時的輔助線之外，還可以使用輔助線上的節點來變更輔助線的位置和角度，改變消失點的位置。

### 在透視尺規上進行描繪

在透視尺規上進行描繪在透視尺規上進行描繪
開啟『對齊到尺規』的[對齊到特殊尺規]，可以使用[沾水筆]或[鉛筆]等繪圖工具，在CLIP STUDIO PAINT的畫布上繪製符合建立的透視尺規的線條。

若無法畫出符合透視尺規的線條，請確認『設定尺規的顯示範圍』的設定。
在對齊描繪途中改變線條的方向
若描繪的方向和預期中不同，只需將感應筆碰觸在繪圖板上，並返回至起筆的地方，即可取消已描繪的線條並重新描繪。

此外，透視尺規還可以自訂對齊方向。請參照『切換對齊的設定』。

備忘

若拖拽到起始位置仍無法重新繪製線條，可以在[環境設定]對話方塊中設定。關於變更方法，請參照『尺規・單位』的[在透視尺規對齊中返回開始描繪的點後，重新確定方向]。

繪製圖形和折線
使用[圖形]・[分格邊框]・[對白框]・[選擇範圍]等各個工具，可以根據透視尺規繪製圖形。但只有在輔助工具的[輸入處理]設為[圖形]・[連續曲線]・[單位曲線]時才能使用。

此外，若將圖形或折線等對齊到透視尺規上，部分設定將無效。
·[直線]或[折線]只有在繪製的線條交點沿著透視尺規描畫時，才能對齊到柵格。斜向的交點無法對齊。

### 刪除尺規

刪除尺規刪除尺規
以下將介紹如何刪除在CLIP STUDIO PAINT中建立的尺規。也可以僅刪除特定的尺規。
刪除圖層內的所有尺規
從[圖層]面板中選擇想要刪除的尺規圖示。進行以下操作後，可以刪除尺規。
·點選[圖層]面板中的[刪除圖層]。
·選擇[圖層]選單→[尺規・分格邊框]→[刪除尺規]。
·將選擇的尺規拖拽到[圖層]面板的[刪除圖層]上。

刪除特定的尺規
若在一個圖層中建立了多個尺規時，可以只刪除特定的尺規。
使用[物件]輔助工具點選想要刪除的尺規。進行以下操作後，可以刪除所選的尺規。
·選擇[圖層]選單→[尺規・分格邊框]→[刪除尺規]。
·選擇[編輯]選單→[刪除]。
·若已連接鍵盤，請按下[Delete]鍵。

若為輔助線時，也可使用[物件]輔助工具點擊輔助線，將其拖拽到畫布視窗的外側後刪除。
