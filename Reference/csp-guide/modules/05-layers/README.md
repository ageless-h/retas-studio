### 檔案物件圖層【PRO／EX】

檔案物件圖層【PRO／EX】檔案物件圖層【PRO／EX】
檔案物件圖層，是指將圖像檔案讀取至CLIP STUDIO PAINT後，可以像圖像素材圖層一樣使用的圖層。
與一般的圖層不同，原始檔案會保持不變，可以在CLIP STUDIO PAINT中作為圖層進行參照。檔案物件圖層無法直接描繪，但可以開啟讀取的原始檔來進行描繪和編輯。在這裡進行的描繪和編輯內容也會套用到檔案物件圖層中。
建立檔案物件圖層
讀取圖像檔案，並建立檔案物件圖層。另外，也可以將普通的圖層轉換為檔案物件圖層。轉換時，可以將其作為另一個檔案進行保存和編輯。

重要

如果更改包含檔案物件圖層的檔案，或更改作為檔案物件圖層參照的檔案的保存位置，檔案物件圖層將無法正確顯示。

建立檔案物件

顯示對話方塊後，選擇欲作為檔案物件進行讀取的檔案，然後點選[打開]。
可以讀取的圖像檔案包括以CLIP STUDIO FORMAT（副檔名：clip）保存的檔案、BMP、JPEG、PNG、WebP、TIFF、Targa、Adobe Photoshop文件

### 什麼是圖層？

什麼是圖層？什麼是圖層？
在使用CLIP STUDIO PAINT等數位繪圖應用程式繪製插圖或漫畫時，會使用到「圖層」。
圖層就像一張透明膠片。線條跟顏色全部會描繪在這張透明的圖層上。而畫布上看到的樣子，就是所有圖層重疊後的狀態。

將線稿、上色、角色與背景等分開繪製在不同的圖層上，即可分別進行編輯。根據圖層的分類方式，能更有效地進行後續的修正工作，例如將不需要的草稿隱藏起來、保留線稿並重新上色，或者調整角色在背景中的位置等。

設定為參照圖層

### 設定混合模式

設定混合模式設定混合模式
混合模式功能，能用於設定圖層要如何對其下方的圖層疊加顏色。透過混合的顏色，可以獲得各種效果。可以應用於光影的表現以及色彩的濃淡表現。

要設定混合模式，請從[圖層]面板中選擇圖層，然後從[圖層]面板的混合模式中選擇所需的效果。圖層的不透明度旁邊會顯示所選的混合模式名稱。
如果選擇了多個圖層，可以同時設定混合模式。但是每個圖層只能設定一種混合模式。

混合模式的種類

### 光澤圖層【EX】
