# 更新歷史記錄
### Ver. 4.0的更新
基於2024年4月的產品程式，進行以下項目的更新。
·於Android中追加『檔案選擇器對話方塊』。以CLIP STUDIO PAINT開啟或保存檔案時，可選擇共有儲存空間中的檔案。
·變更『工具面板』的工具配置。將使用頻率高的筆刷系工具往上移。
·可使用『輔助工具面板』選擇多個輔助工具。除了使用修飾鍵選擇之外，另追加了[使用核取方塊選擇]，會在輔助工具上顯示勾選框，可進行選擇。
·可統整多個輔助工具，一併寫出檔案。可選擇個別輔助工具的寫出，或作為輔助工具群組進行寫出。詳情請參照『寫出多個輔助工具』和『寫出輔助工具組』。
·[移動圖層]工具的輔助工具已移至[操作]工具中。
·於『選擇向量圖層時的輔助工具詳細面板』中追加『修正』類別。
·於[線修正]輔助工具的[輔助工具詳細]面板中追加『修正』類別。
·於[檢視]選單→[對齊]中追加『控制點之間的對齊』。可進行控制點之間的對齊。
·於『選擇漸層圖層時的輔助工具詳細面板』中追加『修正』類別。
·於『選擇檔案物件圖層時的輔助工具詳細面板』中追加『修正』類別。
·可使用[新建]對話方塊等的[記錄縮時攝影]選擇記錄方法。詳情請參照『縮時攝影』。
·於『新建對話方塊（動畫）【PRO／EX】』中追加[動畫作畫紙張設定]。開啟後，可以設定[演出畫格]與[作畫尺寸]等動畫作畫紙張的設定。
·在『新建對話方塊（動畫）【PRO／EX】』中，於[話數]中追加勾選框。可切換從作品資訊到話數的顯示。
·從『從照片圖庫新建【iPad】』與『在儲存空間中新建【Android】』讀取照片後，可提取線稿。
·可從『從照片圖庫新建【iPad】』與『在儲存空間中新建【Android】』讀取影片檔，並新建時間軸為有效的畫布。
·於[檔案]選單中追加『從影片新建【Windows／macOS】』。可讀取影片檔，並新建時間軸為有效的畫布。
·於『寫出縮時攝影對話方塊』與『寫出設定對話方塊』中追加[雜訊圖樣]。可使用『浮水印設定對話方塊』設定雜訊圖樣。
·執行『變更畫布基本設定』時，可使用[畫布基本設定]對話方塊追加或刪除裝訂用的框架與動畫用的框架。
·於『輔助預覽面板』圖像一覽中追加命令列。可於圖像中顯示勾選框，移動或刪除多個圖像。
·追加『工具滑桿面板』。可使用滑桿設定所選擇的輔助工具的筆刷尺寸與不透明度。
·於『顏色設定對話方塊』中追加[預覽]。開啟後，會在畫布上顯示指定顏色的預覽。
·於[選擇範圍]工具中追加『沿著線稿選擇輔助工具』。
·於選擇『套索選擇輔助工具』與『收縮選擇輔助工具』時的[輔助工具詳細]面板中追加『套索』類別。
·於[編輯]選單→[變形]中追加『人偶變形【PRO／EX】』。可以針對所選圖層上的描繪內容建立三角形的網格，並在網格上設定圖釘，進行插圖姿勢變更等修改。可在[輔助工具詳細]面板中追加的『網格・圖釘【PRO／EX】』類別，進行詳細的設定。
·於[檢視]選單→[對齊]中追加『物件之間的對齊』。可進行物件之間的對齊。
·於[檢視]選單→[對齊]中追加『對齊時顯示輔助線』。可在移動物件・節點・控制點時顯示輔助線。
·於[濾鏡]選單中追加『色彩濾鏡』與『濾鏡庫』。可以更直覺性地進行操作，在預覽的同時，使用色彩濾鏡調整顏色或添加模糊等效果的濾鏡。
·於[濾鏡]選單→[模糊]中追加『鏡頭模糊』。會建立圓形模糊或六角形模糊等，重現以攝影機拍攝時自然的模糊效果。
·於[濾鏡]選單→[效果]中追加『水晶』。可以呈現水晶或冰等硬質物體閃閃發光的質感。
·重整『圖層LT轉換對話方塊』的介面，可實現更直覺性地操作。並且可將常用的設定登記為預設。
·於選擇『圍住塗抹輔助工具』時的[輔助工具詳細]面板中追加『套索』類別。
·[塗抹剩餘部分]輔助工具的名稱已變更為『間隙填充筆輔助工具』。
·『套索塗抹輔助工具』已自[圖形]工具移至[填充]工具中。
·於選擇『套索塗抹輔助工具』時的[輔助工具詳細]面板中追加『套索』類別。
·於『輸入・編輯文字時的輔助工具詳細』中追加『修正』類別。移動文字時可以對齊物件或尺規等。
·於[檢視]選單中追加[對齊]。[對齊到尺規]・[對齊到特殊尺規]・[對齊到柵格]・[切換要對齊的特殊尺規]已移至[對齊]中。詳情請參照『對齊到尺規』・『切換要對齊的特殊尺規』。
·物件與控制點也可對齊至尺規。詳情請參照『使物件對齊到尺規』・『使控制點對齊到尺規』。
·[分格邊框]工具的輔助工具已移至[漫畫]工具的『分格邊框群組』中。詳情請參照『分格邊框群組』。
·[分格邊框]工具的[剪下分格邊框]群組已移至[漫畫]工具的[分格邊框]群組中。詳情請參照『使用輔助工具進行分割』。
·[對白框]工具已移至[漫畫]工具的[對白框]群組與[效果線]群組中。詳情請參照『製作對白框』。
·[圖形]工具的[流線]群組與[集中線]群組已移至[漫畫]工具。詳情請參照『製作流線』・『製作集中線』。
·使用『一併寫出』寫出多頁作品時，可以設定[浮水印]與[雜訊圖樣]。
·廢除[檔案]選單→[寫出多頁]→[Kindle格式輸出設定]・[以Kindle格式輸出]・[預覽已輸出的Kindle數據]。適用於Kindle的格式可從『輸出EPUB數據』寫出。
·於[時間軸]面板中追加[顯示面板顏色]。選擇[在片段上顯示面板顏色]後，即可在片段上顯示面板顏色。選擇[在膠片上顯示面板顏色]後，即可在片段中的膠片上顯示面板顏色。
·於『透光桌輔助工具的輔助工具詳細面板』中追加『修正』類別。
·追加2D攝影機資料夾軌道時，可以使用[2D攝影機資料夾]對話方塊設定2D攝影機資料夾的名稱，還可以設定用[基準尺寸]寫出影片的[寬度]與[高度]。詳情請參照『追加2D攝影機資料夾軌道』。
·可從[時間軸]面板的關聯選單建立中割軌道標籤。詳情請參照『建立中割軌道標籤』。
·可從[時間軸]面板的關聯選單中刪除軌道標籤。詳情請參照『刪除軌道標籤』。
·可以使用[檔案]選單→[寫出動畫]→『連號圖像』與『輸出動畫膠片【EX】』設定[檔案名稱的開頭]與[檔案名稱的末尾]。
·使用[檔案]選單→[寫出動畫]→『連號圖像』與『輸出動畫膠片【EX】』保存支援色彩設定檔的格式時，可以嵌入ICC設定檔。
·於[檔案]選單→[寫出動畫]→『輸出動畫膠片【EX】』中追加[輸出動畫資料夾外的圖層]。可將動畫資料夾外的圖層
