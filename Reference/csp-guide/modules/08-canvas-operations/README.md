### 操作的撤銷・重做

操作的撤銷・重做操作的撤銷・重做
以下將介紹在CLIP STUDIO PAINT中進行撤銷（undo）和重做（redo）操作的方法。另外，使用[歷史記錄]面板，可以一次撤銷或重做多個操作記錄。
撤銷
進行以下操作，可以撤銷錯誤的操作。

·點擊命令列的[撤銷]按鈕。
·選擇[編輯]選單→[撤銷]。
·按住[Ctrl]+[Z]鍵。
·如果裝置支援觸控手勢，請使用2根手指點擊。
重做
進行以下操作，可以重新執行已撤銷的操作。

·點擊命令列的[重做]按鈕。
·選擇[編輯]選單→[重做]。
·按下[Ctrl]+[Y]鍵。
·如果裝置支援觸控手勢，請使用3根手指點擊。
歷史記錄面板【PRO／EX】
[歷史記錄]面板是記錄和管理操作歷史記錄的面板。點擊面板上顯示的操作，可以將操作內容恢復到該時間點。

記錄操作的上限可以在[環境設定]對話方塊中進行設定。詳情請參照『性能』。
若想顯示歷史記錄面板
在Windows／macOS／平板電腦上，若想切換[歷史記錄]面板的顯示，請選擇[視窗]選單→[歷史記錄]。
刪除操作的歷史記錄
選擇[編輯]選單→[清除記憶體]→[歷史記錄]，即可從[歷史記錄]面板中

畫布控制器畫布控制器
CLIP STUDIO PAINT畫布視窗左下方的按鈕和滑桿。可以變更畫布的顯示方式。選擇[檢視]選單→[捲軸]，可以切換顯示。

①放大・縮小滑桿拖拽滑桿，可以調整畫布的顯示尺寸。
②縮小詳情請參照『縮小』。
③放大詳情請參照『放大』。
④100%能以原尺寸顯示畫布。
⑤旋轉滑桿拖拽滑桿可以旋轉畫布。
⑥左旋轉可以按照固定的角度刻度將畫布向左旋轉。角度刻度在初始設定中為5度。可以在[環境設定]對話方塊的[顯示角度]中更改角度。詳情請參照『畫布』。
⑦右旋轉可以按照固定的角度刻度將畫布向右旋轉。角度刻度在初始設定中為5度。可以在[環境設定]對話方塊的[顯示角度]中更改角度。詳情請參照『畫布』。
⑧重置旋轉點擊後，可以重置畫布的旋轉。
