# 變形・對齊・均分

·​變形圖像
·​變形方法的種類
·​物件對齊
·​歪斜工具
·​對齊・均分【PRO／EX】
- 變形・對齊・均分

## 详细内容

### 變形方法的種類

變形方法的種類變形方法的種類
選擇CLIP STUDIO PAINT[編輯]選單→[變形]中的項目時，可以執行的變更方法。

備忘

當『物件之間的對齊』為有效時，使用節點進行變形，即可對齊操作中節點以外的節點。[網格變形]的節點也相同。詳情請參照『對齊目標的物件』。

放大・縮小
選擇[編輯]選單→[變形]→[放大・縮小・旋轉]或[放大・縮小]。拖拽節點，可以在保持長寬比的情況下，放大或縮小圖像。
若建立選擇範圍，可以透過選擇範圍啟動器執行 [放大・縮小・旋轉]。

在輔助線下方的變形啟動器中點擊[確定]，即可確定變形。
修飾鍵
變形途中使用修飾鍵，可以暫時變更變形方法。
·按住[Alt]鍵並拖拽節點，可以根據基準點進行放大・縮小。
·按住[Shift]鍵並拖拽節點，可以在固定長寬比的情況下進行放大・縮小。
·按住[Ctrl]鍵並拖拽節點，可以切換為[自由變形]。
·按住[Shift]+[Ctrl]鍵並拖拽節點，可以切換為[歪斜]。
支援的圖層
平塗圖層・色調補償圖層不支援[放大・縮小・旋轉]和[旋轉]。
旋轉
選擇[編輯]選單→[變形]→[放大・縮小・旋轉]或[旋轉]，可以旋轉圖像

### 物件對齊

物件對齊物件對齊
物件對齊是在變形或移動物件時，能對齊其他物件的功能。可以讓物件之間均勻配置，或者進行對齊使相同大小。

備忘

按住[Ctrl]鍵，同時移動或變形物件，即可暫時解除對齊。

物件之間的對齊
選擇[檢視]選單→[對齊]→[物件之間的對齊]後，即可切換物件之間的對齊是否有效。
將[物件之間的對齊]設為有效後，會執行以下的物件配置。
·將多個物件中間對齊、上對齊或下對齊。
·將多個物件等間隔排列。
·當進行變形時，讓物件的寬度或高度與其他物件保持一致。
此外，以『變形圖像』的操作進行變形時，可以讓節點對齊其他物件，或者讓節點之間對齊。

無法對齊時
若物件之間無法對齊，請確認以下設定。
·請開啟[物件]輔助工具等的[工具屬性]面板或者[輔助工具詳細]面板的[依據檢視選單對齊]。詳情請參照『修正』類別。
·有可能正在操作不支援對齊的物件。請參照以下的『可對齊的物件』與『對齊目標的物件』。
可對齊的物件
操作如下的圖層或節點等時，可以進行物件之間的對齊。
向量圖層・對白框圖層・分格邊框資料夾
使用[物件]輔助工具拖拽移動物件時，會進行對齊。執行以拖拽變形物件周圍的節點等操作時，

### 變形圖像

變形圖像變形圖像
使用CLIP STUDIO PAINT進行圖像變形的方法。
選擇想要變形的圖層，點選[編輯]選單→[變形]中的任一項目，即可顯示輔助線和節點。透過操作，能對圖像進行變形。
·圖層中的描繪範圍將成為變形的對象。
·如果已經建立了選擇範圍，則只會對選擇範圍內進行變形。使用選擇範圍進行的變形，可以應用在點陣圖層、向量圖層、圖層蒙版和選擇範圍圖層上。
·如果選擇了多個圖層，可以對所有選取的圖層進行變形。部分變形方法可能不支援圖層功能。

①節點配置在變形範圍上的透明四角。進行拖拽可以變形圖像。節點能拖拽的方向取決於變形方式。變形方法請參照『變形方法的種類』。
②輔助線框起顯示變形範圍的線條。若使用[網格變形]，會顯示網格狀的線。
③基準點詳情請參照『基準點』。
操作圖像素材圖層和透光桌圖層時，可能會稱為中心點。
④變形啟動器能進行變形的確定和取消。詳情請參照『變形啟動器』。

備忘

·變形時，除了操作節點和輔助線之外，還可以使用[工具屬性]面板進行變形。詳情請參照『變形時的工具屬性面板』。
·選擇『人偶變形【PRO／EX】』時，不會顯示節點・輔助線・基準點。

變形啟動器


### 歪斜工具

歪斜工具歪斜工具
使用CLIP STUDIO PAINT的[歪斜]工具，只需要用筆輕輕畫過，就能夠變更特定部分的形狀和尺寸。調整厚塗插圖和底稿的平衡時非常方便。
在[工具]面板中選擇[歪斜]工具即可使用。
與濾鏡不同，[歪斜]工具可以直接在畫布的圖層上進行編輯。適用於點陣圖層、選擇範圍圖層和圖層蒙版。

備忘

在部分環境中，[歪斜]工具可能會被登記為[色彩混合]工具的輔助工具。

在[圖層]面板中，選擇想編輯的圖層。選擇多個圖層，可以統一套用效果。
·選擇包含圖層蒙版的圖層時，可以同時將效果套用至圖層蒙版和圖層兩者。但如果關閉[在圖層上連接蒙版]，則只能對其中一方進行效果的套用。使用[歪斜]工具前，請從圖層和圖層蒙版中選擇要套用效果的項目。
·若選擇圖層資料夾，可以將效果套用至整個圖層資料夾。
在畫布上拖拽或長按想編輯的部分，即可變更畫布內的描繪內容。

選擇歪斜工具時的輔助工具詳細面板
[歪斜]工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定，可以調整歪斜強度和範圍。
在[輔助工具詳細]面板上，可以透過『筆刷尺寸』・『歪斜』・『修正』的類別進行設定。
