## 详细内容

### 填充工具

填充工具填充工具
使用CLIP STUDIO PAINT的[填充]工具，可以填充在畫布上被線條圍起來的區域或者相同顏色的區域。[填充]工具可以從[工具]面板中選擇。
先選好要填充顏色，再從[圖層]面板中選擇要填充的圖層。

備忘

·目前選擇中的顏色就是用來填充的顏色。關於選擇顏色的方法，請參照『選擇・建立顏色』。
·若填充處理花太多時間，可以按下[Esc]鍵，取消處理。

[填充]工具中包含『僅參照編輯圖層輔助工具』・『參照其他圖層輔助工具』・『圍住塗抹輔助工具』・『間隙填充筆輔助工具』・『套索塗抹輔助工具』。可以配合用途進行填充。
僅參照編輯圖層輔助工具
點擊畫布中要填充的位置，可以參照編輯中的圖層進行填充。

如欲一次填充連續的多個封閉區域，請拖拽要填充的範圍。

使用[工具屬性]面板和[輔助工具詳細]面板，可以進行更詳細的設定。更改設定後，即可指定在填充時參照的圖層，也可封閉間隙並進行填充。
在[輔助工具詳細]面板上，可以從『填充』・『參照對象』・『墨水』・『消除鋸齒』的各個類別中設定填充方法。

點擊畫布中要填充的位置，可以參照所有圖層進行填充。僅有參照的

### 套索塗抹輔助工具

套索塗抹輔助工具套索塗抹輔助工具
CLIP STUDIO PAINT的[套索塗抹]輔助工具可以在拖拽畫布並繪製線條的方式下繪製圖形，並填充圖形的內部。選擇[工具]面板的[填充]工具後，可以在[輔助工具]面板中選擇[套索塗抹]輔助工具。
如下圖般搭配使用選擇範圍，或在[輔助工具詳細]面板中設定『套索』類別，也能沿著線稿填色。

透過使用[工具屬性]面板和[輔助工具詳細]面板，可以進行[套索塗抹]輔助工具的更詳細設定。更改設定後，可以調整繪製圖形時的抖動修正和圖形的消除鋸齒等。
在[輔助工具詳細]面板上，可以透過『套索』・『墨水』・『消除鋸齒』・『修正』的各類別設定填充。

### 填充

填充填充

·建立選擇範圍後，點擊選擇範圍啟動器的[填充]，也可以進行相同的操作。關於選擇範圍啟動器，請參照『選擇範圍啟動器』。

### 漸層工具

漸層工具漸層工具
CLIP STUDIO PAINT的[漸層]工具可以進行漸層的描繪。[漸層]工具可以從[工具]面板中選擇。

在畫布想建立漸層的位置上進行拖拽。根據拖拽的位置和長度，將繪製相應的漸層。

[漸層]工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定後，可以追加用於漸層的顏色，並且可以更改漸層的形狀。
在[輔助工具詳細]面板上，可以從『漸層』・『墨水』・『消除鋸齒』的各個類別中設定漸層。
建立漸層圖層【PRO／EX】
在選擇[漸層]工具時，於[工具屬性]面板或[輔助工具詳細]面板中，將[描繪對象]設定為[建立漸層圖層]，即可建立漸層圖層。

以[物件]輔助工具選取漸層圖層，即可在描繪後更改漸層的角度和顏色等設定。關於漸層圖層，請參照『漸層圖層【PRO／EX】』。

備忘

[漸層]工具中部分的輔助工具，從一開始就將[描繪對象]設定為[建立漸層圖層]。在描繪漸層的同時，會建立漸層圖層。

### 等高線填充工具【PRO／EX】

等高線填充工具【PRO／EX】等高線填充工具【PRO／EX】
使用CLIP STUDIO PAINT的[等高線填充]工具，可以透過點擊被2種顏色的線包圍的區域，建立平滑的漸層效果。在[工具]面板上選擇[漸層]工具後，再從[輔助工具]面板中選擇[等高線填充]群組即可使用。
輪廓線加上消除鋸齒後，有可能會讓[等高線填充]工具無法成功填充。建立輪廓線時，請將繪圖工具的[消除鋸齒]設定為[關閉]或[無]。

[等高線填充]工具可以使用[工具屬性]面板和[輔助工具詳細]面板進行更詳細的設定。更改設定後，可以調整使用[等高線填充]工具建立的漸層色階，設定參照的顏色。
在[輔助工具詳細]面板上，可以從『等高線填充』・『填充』・『參照對象』的各個類別中設定填充方法。
