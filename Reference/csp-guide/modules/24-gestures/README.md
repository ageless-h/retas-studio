### 切換到適合觸控手勢的畫面

切換到適合觸控手勢的畫面切換到適合觸控手勢的畫面
若裝置為支援觸控手勢的Windows環境，則可切換到適合觸控手勢的畫面。例如：按鈕顯示變大以便用手指操作，或者收納面板以擴大畫布上可描繪的範圍等。
可以從[環境設定]對話方塊的『介面』的[設定觸控操作]進行設定。

平板電腦也可以進行相同的操作，但在首次啟動時會顯示適合觸控手勢的畫面。
依照環境不同，即使在Windows環境下也有可能會於首次啟動時顯示適合觸控手勢的畫面。

### 切換工具操作・觸控操作

切換工具操作・觸控操作切換工具操作・觸控操作
點擊命令列上的[以手指和筆使用不同工具]，在操作畫布時，可以切換在用一根手指滑動螢幕時的操作。
·開啟後，可以使用一根手指在螢幕上滑動，執行在[設定修飾鍵]對話方塊中設定的操作。初始設定中可以使用[手掌]工具。
·關閉後，可以使用一根手指在螢幕上滑動，來操作選擇中的工具。

點擊[以手指和筆使用不同工具]右側的部分，可以進行觸控手勢相關的設定。
·選擇[設定以手指使用的工具]後，會顯示『設定修飾鍵【PRO／EX】』，即可設定以手指使用的工具。
·選擇[觸控手勢設定]，會顯示[環境設定]對話方塊，即可設定觸控手勢的操作。關於設定項目，請參照『觸控手勢』。

### 執行選單

執行選單執行選單
Windows和平板電腦的情況下，即使隱藏CLIP STUDIO PAINT的選單列，也可以從命令列執行選單。當想要使用較大範圍畫面進行創作時非常方便。
Windows
用手指點擊命令列的[顯示應用程式選單]後，即可使用按鈕來操作選單。

點擊每個按鈕即可選擇選單。依照點擊項目的不同，可以執行選單命令或顯示輔助選單。若要返回上一個項目，請點擊選單畫面左上方的按鈕。

使用與目前使用中的裝置相容的感應筆或滑鼠，點擊[顯示應用程式選單]後，會以普通的選單形式顯示。
平板電腦
點擊命令列中的[顯示應用程式選單]，即可顯示選單。可以執行選單列的各個選單命令。

### 透過觸控手勢操作畫布

透過觸控手勢操作畫布透過觸控手勢操作畫布
以下將介紹使用觸控手勢操作CLIP STUDIO PAINT畫布的方法。可進行畫布的放大、縮小、旋轉等操作。
在簡約模式和工作室模式中皆可使用觸控手勢來操作畫布。
移動畫布的位置
用兩指滑動畫布，即可移動畫布的位置。

設定[以手指和筆使用不同工具]後，可以使用一根手指滑動來移動畫布的位置。關於設定[以手指和筆使用不同工具]的方法，請參照『以手指和筆使用不同工具』。
旋轉畫布
用兩指旋轉畫布，可以將畫布旋轉到更容易繪畫的角度。

縮放畫布
用兩指向外撥動畫布，可以將畫布放大顯示。
用兩指向內撥動畫布，可以縮小畫布。

操作的撤銷・重做
用兩指點擊畫布，即可取消前一次執行的操作。
用三指點擊畫布，即可重新執行已取消的操作。

獲取顏色
若想使用已繪製在畫布中內容的顏色，請用一根手指長按該部分。

變更設定
可以在[環境設定]對話方塊中設定觸控手勢。詳情請參照『觸控手勢』。
Windows環境中，可以在『觸控手勢』中選擇以[Windows模式]或[CLIP STUDIO模式]執行觸控手勢的方法。選擇[Windows模式]後，會使用按住點擊取代三指點

### 以觸控手勢操作面板

以觸控手勢操作面板以觸控手勢操作面板
能使用觸控手勢操作CLIP STUDIO PAINT面板。
·長按面板即可顯示關聯選單。根據長按的位置，可以進行各種操作。有些面板可能無法顯示關聯選單。[工具屬性]面板和[輔助工具詳細]面板無法顯示關聯選單。
·用兩指點擊含有命令列和屬性列的面板時，可以切換命令列的顯示或隱藏。

備忘

·在可排序按鈕的面板上，若將[排序方法]設定為[Ctrl+拖拽]，會無法僅以滑動來移動按鈕。
·若想執行切換面板的命令列和屬性列的顯示操作，需要在[環境設定]對話方塊的[觸控手勢]中，將[面板]的[兩指點擊]設定為[顯示・隱藏屬性・命令列]。初始設定為[無效]。

快速存取面板
快速存取設定按鈕的順序
左右滑動快速存取組或組內的按鈕後，可以在[快速存取]面板內，移動快速存取組或組內按鈕的位置。移動按鈕時，請勿長按按鈕。

移動工具
上下滑動或左右滑動工具後，即可在[工具]面板內移動工具的位置。移動工具時，請勿長按工具。

### 使用側邊鍵盤進行操作

使用側邊鍵盤進行操作使用側邊鍵盤進行操作
側邊鍵盤是用來輔助CLIP STUDIO PAINT操作的功能。使用側邊鍵盤，可以執行修飾鍵的操作。另外，若在『捷徑鍵設定』設定側邊鍵盤的按鍵，即可作為快捷鍵使用。

備忘

Wiodows平板電腦及不支援觸控手勢的液晶繪圖板，無法使用側邊鍵盤進行操作。

iPad 
會顯示macOS基準的按鍵。可以像鍵盤上的其他按鍵一樣使用。

①esc②control③Shift④option⑤command⑥space⑦觸控鍵有1～15個按鍵。依照iPad顯示器的大小不同，顯示的按鍵數量會有所不同。可用於設定快捷鍵等。只需點擊按鍵，即可執行命令或自動動作等操作。
Android 
會顯示Windows基準的按鍵。可以像鍵盤上的其他按鍵一樣使用。

①Esc②Ctrl③Shift④Alt⑤Space⑥觸控鍵有1～15個按鍵。依照繪圖板顯示器的大小不同，顯示的按鍵數量會有所不同。可用於設定快捷鍵等。只需點擊按鍵，即可執行命令或自動動作等操作。
