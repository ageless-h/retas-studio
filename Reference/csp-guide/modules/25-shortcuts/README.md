### 操作中可使用的修飾鍵

操作中可使用的修飾鍵操作中可使用的修飾鍵
以下是初始設定狀態下可使用的修飾鍵。在按住按鍵的期間，可以暫時切換成別的動作。
除了共通的修飾鍵外，還有可用於每個工具的修飾鍵。共通修飾鍵和工具修飾鍵重複時，會以工具修飾鍵為優先。欲變更設定請參照『設定修飾鍵【PRO／EX】』。
另外，使用工具時，若搭配其他各面板的捷徑鍵來運用修飾鍵，可以更有效地使用工具。面板的捷徑鍵請參照『選項的捷徑鍵』。

備忘

除非特別註明，修飾鍵皆指Windows的按鍵。macOS／iPad的用戶，請將[Alt]鍵替換成[Option]鍵、[Ctrl]鍵替換成[Command]鍵使用。
