## 详细内容

### 工具面板

工具面板工具面板
CLIP STUDIO PAINT的[工具]面板中有各種工具以按鈕形式排列。點擊按鈕即可使用工具。

A.工具按鈕
筆刷和橡皮擦等工具的按鈕排列在一起。點擊按鈕即可使用工具。工具按鈕上會顯示所選擇的輔助工具的圖示。因此，初始設定和畫面可能會有所不同。
關於各個工具的詳細內容，請參照『CLIP STUDIO PAINT的工具』。
關於變更工具名稱和重新排列工具配置，請參照『工具面板的自訂設定』。
B.色彩圖示
能夠從主顏色、輔助顏色和透明色中選擇要繪製或設定的顏色。
主顏色和輔助顏色，能在以顯示顏色為灰色的筆刷前端圖像進行描繪時使用，或者在繪製圖形或對白框時進行組合使用。
色彩圖示可以在[工具]面板選單的[顯示色彩圖示]中，切換顯示或隱藏。
①主顏色也稱為主描繪色。點擊主顏色的圖示，即可在選色的面板中設定主顏色。另外，描繪系工具也可以使用主顏色進行描繪。
②輔助顏色也稱為輔助描繪色。點擊輔助顏色的圖示，即可在選色的面板中設定輔助顏色。另外，描繪系工具也可以使用輔助顏色進行描繪。
③透明色點擊即可使用透明色進行描繪。無法進行顏色設定。

### 輔助工具面板

輔助工具面板輔助工具面板
CLIP STUDIO PAINT的每個工具都具備預設設定，這些設定被稱為「輔助工具」。
輔助工具被分配到[輔助工具]面板的按鈕上。點擊後，將切換[工具屬性]面板的設定。某些工具被分成多個群組，切換群組後可以顯示其他輔助工具。

①群組切換如果有多個群組工具，可以切换工具的群組。
②輔助工具包含在群組中的輔助工具一覽。可以在這裡選擇輔助工具。
一邊按住[Ctrl]鍵或[Shift]鍵一邊進行選擇，即可選取多個輔助工具。可以一併移動或寫出多個輔助工具。 
③追加輔助工具會顯示『追加素材對話方塊』，可以讀取從CLIP STUDIO ASSETS下載的素材。詳情請參照『讀取素材的方法』。
④使用核取方塊選擇開啟後，可以顯示[輔助工具]面板中各個輔助工具的核取方塊。在想選擇多個輔助工具時十分方便。詳情請參照『在輔助工具上顯示勾選框』。 
⑤追加輔助工具會顯示『追加素材對話方塊』，可以讀取從CLIP STUDIO ASSETS下載的素材。詳情請參照『讀取素材的方法』。
⑥複製輔助工具會顯示[複製輔助工具]對話方塊，並複製目前選擇中的輔助工具。詳情請參照『複製輔助工具』。

### CLIP STUDIO PAINT的工具

CLIP STUDIO PAINT的工具CLIP STUDIO PAINT的工具
CLIP STUDIO PAINT的[工具]面板中初始登記的工具一覽。可以在下方找到每個工具的詳細說明。

A.沾水筆工具
可以像蘸水筆一樣繪製出有強弱的線條，也可以像萬能筆般繪製出粗細均勻的線條。詳情請參照『沾水筆工具』和『點筆輔助工具』。
B.鉛筆工具
可以如使用鉛筆或粉彩等筆觸般，進行有濃淡變化的描繪。詳情請參照『草稿工具』。
C.毛筆工具
可以如使用毛筆般，用混合多種顏色的筆觸進行描繪。詳情請參照『毛筆工具』。
D.橡皮擦工具
可刪除已繪製的內容。詳情請參照『橡皮擦工具』。
E.噴槍工具
可以如使用噴槍般，以帶有模糊效果的筆觸進行描繪。詳情請參照『噴槍工具』。
F.裝飾工具
可用於繪製背景和效果等的圖案。詳情請參照『裝飾工具』。
G.色彩混合工具
將顏色像暈染般混合在一起。詳情請參照『色彩混合工具』。
H.歪斜工具
扭曲液化調整圖像。詳情請參照『歪斜工具』。
I.選擇範圍工具
[選擇範圍]工具可以透過圖形或描繪，來建立各種選擇範圍。詳情請參照『選擇範圍工具』。

### 復原初始工具

復原初始工具復原初始工具
可以恢復安裝CLIP STUDIO PAINT時的工具和輔助工具。也可追加已刪除的初始設定工具。

備忘

在Ver. 1.13.2之前已經安裝CLIP STUDIO PAINT的狀況下，可能會有無法從[追加初始輔助工具]還原某些輔助工具的情況發生。這些筆刷可以從CLIP STUDIO ASSETS下載。
･Ver. 1.10.9之前的初始輔助工具
·Ver. 1.13.2之前的初始輔助工具

工具
從[工具]面板的選單中選擇[追加初始工具]。從[追加初始工具]對話方塊中選擇工具，然後點擊[OK]。所選的工具將追加至[工具]面板中。
輔助工具
從[工具]面板中選擇想追加的輔助工具。從[輔助工具]面板的選單中選擇[追加初始輔助工具]。從[追加初始輔助工具]對話方塊中選擇輔助工具，然後點擊[OK]。所選的輔助工具將追加至[輔助工具]面板中。

### 工具屬性面板

工具屬性面板工具屬性面板
可以設定在[輔助工具]面板上選取的輔助工具。

①筆劃預覽顯示所選輔助工具的筆劃預覽。[工具屬性]面板和[輔助工具詳細]面板的設定會反映在筆劃預覽中。
②鎖定【PRO／EX】可以將在[工具屬性]面板上設定中的數值，保存到選擇中的輔助工具中。鎖定中可以更改[工具屬性]面板的設定，但如果再次選擇相同的輔助工具，將返回到設定為[鎖定]時的狀態。鎖定的輔助工具會在[輔助工具]面板上顯示圖示。
從[輔助工具]面板和[工具屬性]面板的選單中選擇[鎖定]，也可以鎖定輔助工具的設定。
③工具屬性面板[工具屬性]面板的設定畫面。可以對從[輔助工具]面板中選擇的輔助工具進行設定。工具屬性面板上顯示的項目會根據所選擇的輔助工具而有所不同。
④返回初始設定在[工具屬性]面板等地方解除更改的設定，將輔助工具恢復為初始設定。
從[輔助工具]面板和[工具屬性]面板的選單中選擇[返回初始設定]，可以將輔助工具的設定恢復為初始設定。
⑤輔助工具詳細【PRO／EX】可切換[輔助工具詳細]面板的顯示或隱藏。[輔助工具詳細]面板可以進行比[工具屬性]面板更詳細的輔助工具設定。詳情請參照『輔助工具詳細

### 輔助工具詳細面板【PRO／EX】

輔助工具詳細面板【PRO／EX】輔助工具詳細面板【PRO／EX】
可在CLIP STUDIO PAINT的[輔助工具]面板中設定所選的輔助工具。
雖然和[工具屬性]面板具有相同的功能，但[輔助工具詳細]面板可以設定[工具屬性]面板上未顯示的項目。
[輔助工具詳細]面板可能會根據顯示面板的尺寸而有所不同。

①筆劃預覽顯示所選輔助工具的筆劃預覽。[工具屬性]面板和[輔助工具詳細]面板的設定會反映在筆劃預覽中。
②類別選擇類別後，設定輔助工具將顯示不同的項目。可選擇的類別會依照所選的輔助工具而有所不同。
如果[輔助工具詳細]面板的寬度較窄，可以從面板上方的下拉選單中選擇類別。另外，也可以點擊下拉選單左右的按鈕來切換類別。
當[輔助工具詳細]面板的縱寬無法容納類別的設定項目時，類別名稱後面會加上連號，並將類別分割。
③顯示在工具屬性可以設定在[工具屬性]面板中顯示的項目。開啟後，當選擇輔助工具時，會顯示設定在[工具屬性]面板中的項目。
④顯示選單點選後會顯示選單。可設定的項目與[工具屬性]面板相同。
關於[複製值]，請參照『複製值』。關於其他項目，請參照『顯示滑桿和顯示指示器【PRO／EX】
