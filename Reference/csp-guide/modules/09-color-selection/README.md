### 顏色設定對話方塊

顏色設定對話方塊顏色設定對話方塊
[顏色設定]對話方塊，是在使用CLIP STUDIO PAINT進行顏色選擇操作時顯示的畫面。可以像在面板上選色般、輸入數值變更顏色或者是從畫面中取的顏色。

①原本的顏色開啟[顏色設定]對話方塊後，會顯示當前的設定顏色。點擊即可回到原本的顏色。
②設定的顏色在[顏色設定]對話方塊中，將會顯示選擇的顏色。開啟[顏色設定]對話方塊後，不會立即顯示。
③獲取畫面的顏色【Windows／macOS】點擊後，滑鼠游標會切換為吸管。可以使用吸管工具參照畫面上的顏色並選擇顏色。
④設定方法從[標準]、[色環1]、[色環2]、[色板]、[顏色歷史記錄]中，可以選擇設定顏色的方法。將根據選擇的內容切換[顏色設定]對話方塊的顯示。
若使用DEBUT，無法選擇[顏色歷史記錄]。
⑤色相環將色相環切換為[HLS]或[HSV]之一。從[設定方法]中選擇[標準]、[色環1]或[色環2]，即可顯示。
⑥預覽設為開啟後，畫布上會顯示指定顏色的預覽。若功能不支援預覽，不會顯示此項目。
⑦顏色色域確認實際顏色並選擇顏色的畫面。依據所選擇的[設定方法]，顯示會有所不同。

### 色環面板

色環面板色環面板
CLIP STUDIO PAINT的[色環]面板，可以透過拖拽外圈的色相環來選擇大致的顏色，並從中央的色彩區域中選擇描繪顏色。
選擇的顏色將顯示在色環左下方的色彩圖示上。另外，所選顏色的數值將以數字形式顯示在色環的右下方。點擊數值部分後，可以從顯示在色環中的色彩空間值切換到RGB值。

點擊面板右下的圖示即可切換色彩空間。
此外，也可以從[色環]面板的選單中，選擇切換要顯示的色彩空間。

①HSV色彩空間可以透過調整色相（Hue）、彩度（Saturation）和明度（Value）來設定顏色。
②HLS 色彩空間調整色相（Hue）、輝度（Luminance）、彩度（Saturation）可以設定顏色。
色彩圖示
能夠從主顏色、輔助顏色和透明色中選擇要繪製或設定的顏色。
主顏色和輔助顏色，能在以顯示顏色為灰色的筆刷前端圖像進行描繪時使用，或者在繪製圖形或對白框時進行組合使用。

①主顏色也稱為主描繪色。點擊主顏色的圖示，即可在選色的面板中設定主顏色。另外，描繪系工具也可以使用主顏色進行描繪。
②輔助顏色也稱為輔助描繪色。點擊輔助顏色的圖示，即可在選色的面板中設定輔助顏色。

CLIP STUDIO PAINT的[顏色滑桿]面板，是一個可以透過滑桿或輸入數值來設定描繪色的面板。顏色的設定方法可以在[顏色滑桿]面板的左側選項中進行切換。可以設定的值有RGB、HSV、HLS和CMYK。
設定的顏色會顯示在[顏色滑桿]面板下方的色彩圖示中。
色彩圖示可以在[顏色滑桿]面板中切換顯示或隱藏。關於色彩圖示的功能請參照『色環面板』。
RGB選項
選擇[RGB]選項，可以使用RGB值來設定描繪色。
RGB是光的三原色，表示紅色（Red）、綠色（Green）、藍色（Blue），用於顯示在映像管和液晶顯示器等設備上的顏色。

HSV選項・HLS選項
選擇[HSV]選項後，可以在HSV色彩空間中設定描繪色。如果在[色環]面板中以HLS色彩空間顯示，則會顯示[HLS]選項，可以在HLS色彩空間中設定描繪色。
此外，還可以從[顏色滑桿]面板的選單中選擇要顯示的色彩空間，並切換為HSV色彩空間或HLS色彩空間。

①HSV色彩空間可以透過調整色相（Hue）、彩度（Saturation）和明度（Value）來設定顏色。

### 色板面板

色板面板色板面板
CLIP STUDIO PAINT的[色板]面板，是一個可以從準備了標準顏色的色板中，輕鬆點選顏色的面板。
[色板]面板不只有最初登記的[色板]，還可以自行建立與追加原創色板。

①顯示色板可以從下拉選單中選擇色板。選擇後，將會切換顯示在色板一覽中的顏色。
②編輯色板開啟[編輯色板]對話方塊，可以進行色板的管理。詳情請參照『管理色板』。
③追加色板可以打開『追加素材對話方塊』，讀取從CLIP STUDIO ASSETS下載的色板素材。詳情請參照『讀取素材的方法』。
④色板一覽點擊顏色區塊可選擇描繪色。
若使用Windows／macOS裝置，將滑鼠游標移至顏色區塊上，提示框中會顯示顏色的RGB值或HSV值以及顏色的名稱。
關於自訂顏色的顯示方法，請參照『設定色板的顏色』。
⑤RGB值・HSV值描繪色的RGB值會以數值方式顯示。點擊RGB值，顯示將切換為HSV值。但是，如果[色環]面板的顯示是HLS色彩空間，則會切換為HLS值。
⑥替換顏色請參照『替換顏色』。
⑦追加顏色請參照『追加顏色』。
⑧刪除顏色請參照『刪除顏色』。

### 色彩混合面板

色彩混合面板色彩混合面板
CLIP STUDIO PAINT的[色彩混合]面板，可以在描繪色區域上使用[毛筆]工具或[色彩混合]工具進行描繪和混色。可以在畫布以外的地方試色，或使用吸管工具取得描繪區域的顏色。

①描繪區域可以使用[毛筆]和[色彩混合]等工具，進行描繪和混色的區域。可以透過滑鼠滾輪或者向內撥動・向外撥動來放大・縮小描繪區域。
②色板列可以登記常用的顏色。點擊顏色區塊以選擇顏色，可以將描繪色更改為所選的顏色。拖放區塊可以變更區塊的顯示順序。
關於設定色板顏色的方法，請參照『設定彩色板列的顏色』。
③描繪色區域會顯示選擇中的描繪顏色。點擊即可從『顏色設定對話方塊』選擇顏色。
④清除可刪除描繪區域中的描繪內容。
⑤撤銷可以取消描繪區域中的描繪內容。若使用平板電腦，可以用2指點擊描繪區域進行相同的操作。
⑥重做在描繪區域中，可以重做已撤銷的操作。若使用平板電腦，可以用3指點擊描繪區域進行相同的操作。
⑦筆刷尺寸在描繪區域中，可以從[大]、[中]、[小]中選擇要使用的筆刷尺寸。當[色彩混合]面板的寬度較窄時，將顯示下拉選單而非按鈕。

### 近似色面板【PRO／EX】

近似色面板【PRO／EX】近似色面板【PRO／EX】   
CLIP STUDIO PAINT的[近似色]面板，可以根據在其他面板中選擇的描繪顏色，顯示近似顏色的面板。

顯示近似顏色面板

色彩圖示可以在[顏色滑桿]面板中切換顯示或隱藏。關於色彩圖示的功能請參照『色環面板』。
RGB選項
選擇[RGB]選項，可以使用RGB值來設定描繪色。
RGB是光的三原色，表示紅色（Red）、綠色（Green）、藍色（Blue），用於顯示在映像管和液晶顯示器等設備上的顏色。

HSV選項・HLS選項
選擇[HSV]選項後，可以在HSV色彩空間中設定描繪色。如果在[色環]面板中以HLS色彩空間顯示，則會顯示[HLS]選項，可以在HLS色彩空間中設定描繪色。

色環面板色環面板
CLIP STUDIO PAINT的[色環]面板，可以透過拖拽外圈的色相環來選擇大致的顏色，並從中央的色彩區域中選擇描繪顏色。
選擇的顏色將顯示在色環左下方的色彩圖示上。另外，所選顏色的數值將以數字形式顯示在色環的右下方。點擊數值部分後，可以從顯示在色環中的色彩空間值切換到RGB值。

點擊面板右下的圖示即可切換色彩空間。
此外，也可以從[色環]面板的選單中，選擇切換要顯示的色彩空間。

①HSV色彩空間可以透過調整色相（Hue）、彩度（Saturation）和明度（Value）來設定顏色。
②HLS 色彩空間調整色相（Hue）、輝度（Luminance）、彩度（Saturation）可以設定顏色。
色彩圖示
能夠從主顏色、輔助顏色和透明色中選擇要繪製或設定的顏色。
主顏色和輔助顏色，能在以顯示顏色為灰色的筆刷前端圖像進行描繪時使用，或者在繪製圖形或對白框時進行組合使用。

①主顏色也稱為主描繪色。點擊主顏色的圖示，即可在選色的面板中設定主顏色。另外，描繪系工具也可以使用主顏色進行描繪。
②輔助顏色也稱為輔助描繪色。點擊輔助顏色的圖示，即可在選色的面板中設定輔助顏色。
