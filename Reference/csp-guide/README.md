## 软件概述

CLIP STUDIO PAINT（简称 CSP）是由日本 CELSYS 公司开发的专业绑画和漫画创作软件。支持 Windows、macOS、iPad 和 Android 多平台，是目前最受欢迎的漫画和插画创作工具之一。

### 版本说明

| 版本 | 说明 |
|------|------|
| 免费版 | 基础绑画功能，有使用限制 |
| PRO版 | 专业版，包含大部分绑画和漫画功能 |
| EX版 | 完整版，包含所有功能（多页管理、高级动画、3D功能等） |

### 主要功能领域

1. **绑画创作** - 丰富的笔刷系统和自然的绑画体验
2. **漫画制作** - 专业的分格、网点、对白框功能
3. **动画制作** - 完整的时间轴和关键帧动画系统
4. **3D辅助** - 3D 素描人偶和模型导入功能
