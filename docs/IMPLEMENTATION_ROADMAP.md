# RETAS Studio 实施路线图

## 目标

构建可用于生产环境的 2D 动画制作软件。

## Phase 1: 基础完善 (2-3 周)

**目标**: 让软件可用于简单动画制作

### Week 1: 多图层系统

#### Day 1-2: 核心架构
- [ ] 重构 Layer 结构体
  - 支持多种图层类型 (Raster/Vector)
  - 添加混合模式枚举
  - 添加不透明度属性
- [ ] 创建 LayerManager
  - 图层列表管理
  - Z-index 排序
  - 添加/删除/重命名

#### Day 3-4: UI 实现
- [ ] 图层面板重构
  - 可滚动图层列表
  - 图层可见性复选框
  - 图层锁定图标
  - 不透明度滑块
- [ ] 混合模式下拉菜单
  - Normal
  - Multiply
  - Add
  - Screen

#### Day 5-7: 渲染集成
- [ ] 多层渲染管线
  - 从底到顶渲染
  - 混合模式实现
  - 透明度合成
- [ ] Canvas 更新
  - 显示所有图层
  - 活动图层高亮

**交付物**: 可创建/管理多个图层的系统

| 风险 | 概率 | 影响 | 缓解措施 |
